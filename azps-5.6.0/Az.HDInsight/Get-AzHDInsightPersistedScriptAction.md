# Get-AzHDInsightPersistedScriptAction

## SYNOPSIS
클러스터에 대한 지속 스크립트 작업을 얻게 하여 시간 순서로 나열하거나 지정된 지속형 스크립트 작업에 대한 세부 정보를 얻습니다.

## 구문

```
Get-AzHDInsightPersistedScriptAction [-ClusterName] <String> [[-Name] <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzHDInsightPersistedScriptAction** cmdlet은 Azure HDInsight 클러스터에 대한 영구 스크립트 작업을 얻게 하여 시간 순서대로 나열하거나 지정된 영구 스크립트 작업에 대한 세부 정보를 얻습니다.

## 예제

### 예제 1: 클러스터에서 지속된 스크립트 작업 얻습니다.
```
PS C:\>Get-AzHDInsightPersistedScriptAction -ClusterName "your-hadoop-001"
```

이 명령은 your-hadoop-001이라는 클러스터에서 영구 스크립트 작업을 얻습니다.
