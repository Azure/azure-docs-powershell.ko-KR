# Set-AzOperationalInsightsDataSource

## SYNOPSIS
데이터 원본을 업데이트합니다.

## 구문

```
Set-AzOperationalInsightsDataSource [-DataSource] <PSDataSource> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Set-AzOperationalInsightsDataSource** cmdlet은 데이터 원본을 업데이트합니다.
