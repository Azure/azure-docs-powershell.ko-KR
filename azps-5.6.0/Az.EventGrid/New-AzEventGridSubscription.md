## 설명
Azure Event Grid 토픽, 지원되는 Azure 리소스, Azure 구독 또는 리소스 그룹에 대한 새 이벤트 구독을 생성합니다.
현재 선택한 Azure 구독에 대한 이벤트 구독을 만들하려면 이벤트 구독 이름 및 대상 엔드포인트를 지정합니다.
리소스 그룹에 대한 이벤트 구독을 만들하려면 이벤트 구독 이름 및 대상 엔드포인트 외에도 리소스 그룹 이름을 지정합니다.
Azure Event Grid 토픽에 대한 이벤트 구독을 만들 경우 토픽 이름도 지정합니다.
지원되는 Azure 리소스에 대한 이벤트 구독을 만들하려면 리소스의 전체 리소스 ID를 지정합니다. 지원되는 형식 목록을 보기 위해 Get-AzEventGridTopicType cmdlet을 실행합니다.

## 예제

### 예제 1
```powershell
PS C:\> New-AzEventGridSubscription -ResourceGroup MyResourceGroup -TopicName Topic1 -Endpoint https://requestb.in/19qlscd1 -EventSubscriptionName EventSubscription1
```

웹후크 대상 엔드포인트를 사용하여 리소스 그룹 \` \` \` \` \` MyResourceGroupName에서 Azure Event Grid 토픽 토픽1에 새 이벤트 구독 EventSubscription1을 \` `https://requestb.in/19qlscd1` 만듭니다. 이 이벤트 구독은 기본 필터를 사용합니다.

### 예제 3
```powershell

```

웹후크 대상 엔드포인트를 사용하여 현재 선택한 Azure 구독에 새 이벤트 구독 \` EventSubscription1을 \` `https://requestb.in/19qlscd1` 만듭니다. 이 이벤트 구독은 기본 필터를 사용합니다.

### 예제 4
```powershell
PS C:\> $includedEventTypes = "Microsoft.Resources.ResourceWriteFailure", "Microsoft.Resources.ResourceWriteSuccess"
PS C:\> $labels = "Finance", "HR"
PS C:\> New-AzEventGridSubscription -Endpoint https://requestb.in/19qlscd1 -EventSubscriptionName EventSubscription1 -SubjectBeginsWith "TestPrefix" -SubjectEndsWith "TestSuffix" -IncludedEventType $includedEventTypes -Label $labels
```

웹후크 대상 엔드포인트를 사용하여 현재 선택한 Azure 구독에 새 이벤트 구독 \` EventSubscription1을 \` `https://requestb.in/19qlscd1` 만듭니다. 이 이벤트 구독은 이벤트 유형 및 제목에 대한 추가 필터를 지정하고 해당 필터와 일치하는 이벤트만 대상 엔드포인트로 배달됩니다.

### 예제 5

지정된 이벤트 허브를 이벤트의 대상으로 사용하여 현재 선택한 Azure 구독에 새 이벤트 구독 \` EventSubscription1을 \` 만듭니다. 이 이벤트 구독은 기본 필터를 사용합니다.

```powershell

```

지정된 웹후크 대상 엔드포인트가 있는 \` EventHub 네임스페이스에 EventSubscription1 새 이벤트 \` 구독을 `https://requestb.in/19qlscd1` 만듭니다. 이 이벤트 구독은 기본 필터를 사용합니다.
