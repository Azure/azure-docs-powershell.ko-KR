# Remove-AzManagedApplication

## SYNOPSIS
관리되는 애플리케이션 제거

## 구문

### RemoveByNameAndResourceGroup(기본값)
```
Remove-AzManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzManagedApplication** cmdlet은 관리되는 애플리케이션을 제거합니다.

## 예제

### 예제 1: 리소스 ID로 관리되는 애플리케이션 제거
```
PS C:\>$Application = Get-AzManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplication -Id $Application.ResourceId -Force
```

첫 번째 명령은 Get-AzManagedApplication cmdlet을 사용하여 myApp이라는 관리되는 애플리케이션을 얻습니다.
명령은 이 변수를 $Application 저장합니다.
두 번째 명령은 리소스의 **ResourceId** 속성으로 식별된 관리되는 애플리케이션을 $Application.
