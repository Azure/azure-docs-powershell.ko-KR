# Restore-AzMySqlFlexibleServer

## SYNOPSIS
기존 백업에서 서버 복원

## 구문

```
Restore-AzMySqlFlexibleServer -Name <String> -ResourceGroupName <String> -InputObject <IServerAutoGenerated>
 -RestorePointInTime <DateTime> [-SubscriptionId <String>] [-Zone <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## 설명
기존 백업에서 서버 복원

## 예제

### 예제 1: PointInTime 복원을 사용하여 MySql 서버 복원
```powershell
PS C:\> $restorePointInTime = (Get-Date).AddMinutes(-10)
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | Restore-AzMySqlFlexibleServer -Name mysql-test-restore -ResourceGroupName PowershellMySqlTest -RestorePointInTime $restorePointInTime 

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName   SkuTier       
----               -------- ------------------ ------- ----------------------- -------   -------       
mysql-test-restore eastus   mysql_test         5.7     10240                   Standard_D2ds_v4 GeneralPurpose
```

이러한 cmdlet은 PointInTime 복원을 사용하여 MySql 서버를 복원합니다.

복잡한 매개 변수 속성

아래에 설명된 매개 변수를 만들 경우 적절한 속성을 포함하는 해시 테이블을 생성합니다. 해시 테이블에 대한 자세한 내용은 Get-Help about_Hash_Tables.


INPUTOBJECT : 복원할 원본 <IServerAutoGenerated> 서버 개체입니다.
  - `Location <String>`: 리소스가 있는 지리적 위치
  - `[Tag <ITrackedResourceTags>]`: 리소스 태그입니다.
    - `[(Any) <String>]`: 이 개체에 속성을 추가할 수 있는 경우를 나타냅니다.
  - `[AdministratorLogin <String>]`: 서버의 관리자의 로그인 이름입니다. 서버를 만들 때만 지정할 수 있습니다(및 생성에 필요합니다).
  - `[AdministratorLoginPassword <SecureString>]`: 관리자 로그인의 암호(서버 만들기에 필요).
  - `[AvailabilityZone <String>]`: 서버의 가용성 영역 정보입니다.
  - `[CreateMode <CreateMode?>]`: 새 MySQL 서버를 만드는 모드입니다.
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`: 위임된 서브넷 암 리소스 ID입니다.
  - `[HaEnabled <HaEnabledEnum?>]`: 서버에 HA를 사용하도록 설정하거나 사용 안 함.
  - `[IdentityType <ResourceIdentityType?>]`: ID 형식입니다.
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`: 서버가 인프라 암호화를 사용하도록 설정하는지 여부를 보여주는 상태입니다.
  - `[MaintenanceWindowCustomWindow <String>]`: 사용자 지정 창을 사용하도록 설정하거나 사용하지 않도록 설정되어 있는지 여부를 나타냅니다.
  - `[MaintenanceWindowDayOfWeek <Int32?>]`: 유지 관리 기간의 주일
  - `[MaintenanceWindowStartHour <Int32?>]`: 유지 관리 기간 시작 시간
  - `[MaintenanceWindowStartMinute <Int32?>]`: 유지 관리 기간의 시작 분
  - `[PropertiesTag <IServerPropertiesTags>]`: 키-값 쌍의 형태로 애플리케이션별 메타데이터입니다.
    - `[(Any) <String>]`: 이 개체에 속성을 추가할 수 있는 경우를 나타냅니다.
  - `[ReplicationRole <String>]`: 복제 역할입니다.
  - `[RestorePointInTime <DateTime?>]`: 복원할 시간을 지정하여 지점 만들기 시간(ISO8601 형식)을 복원합니다.
  - `[SkuName <String>]`: sku의 이름(예: Standard_D32s_v3.
  - `[SkuTier <SkuTier?>]`: 특정 SKU의 계층(예: GeneralPurpose)입니다.
  - `[SourceServerId <String>]`: 원본 MySQL 서버 ID입니다.
  - `[SslEnforcement <SslEnforcementEnum?>]`: 서버에 연결할 때 ssl 적용을 사용하도록 설정하지 않습니다.
  - `[StorageProfileBackupRetentionDay <Int32?>]`: 서버에 대한 백업 보존일입니다.
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`: 저장소 자동 증가를 사용하도록 설정합니다.
  - `[StorageProfileStorageIop <Int32?>]`: 서버에 대한 저장소 IOPS입니다.
  - `[StorageProfileStorageMb <Int32?>]`: 서버에 대해 허용되는 최대 저장소입니다.
  - `[Version <ServerVersion?>]`: 서버 버전입니다.
