# Set-AzCdnEndpoint

## SYNOPSIS
CDN 엔드포인트를 업데이트합니다.

## 구문

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Set-AzCdnEndpoint** cmdlet은 CDN(Azure Content Delivery Network) 엔드포인트를 업데이트합니다.
