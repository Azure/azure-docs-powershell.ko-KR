# Get-AzCdnProfile

## SYNOPSIS
CDN 프로필을 얻습니다.

## 구문

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzCdnProfile** cmdlet은 CDN(Azure Content Delivery Network) 프로필 및 관련 정보를 얻습니다.
