# Remove-AzBatchJobSchedule

## SYNOPSIS
Batch 작업 일정을 제거합니다.

## 구문

```
Remove-AzBatchJobSchedule [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzBatchJobSchedule** cmdlet은 Azure Batch 작업 일정을 제거합니다.

## 예제

### 예제 1: Batch 작업 일정 삭제
```
PS C:\>Remove-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context
```

이 명령은 MyJobSchedule ID가 있는 작업 일정을 삭제합니다.
이 명령은 작업을 삭제하기 전에 확인을 묻는 메시지를 표시합니다.
Get-AzBatchAccountKey cmdlet을 사용하여 변수 변수에 컨텍스트를 $Context 합니다.

### 예제 2: 파이프라인을 사용하여 확인 없이 Batch 작업 삭제
```
PS C:\>Get-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context | Remove-AzBatchJobSchedule -Force -BatchContext $Context
```

이 명령은 id MyJobSchedule이 있는 작업 일정을 Get-AzBatchJobSchedule cmdlet을 사용하게 됩니다.
명령은 파이프라인 연산자를 사용하여 현재 cmdlet에 작업 일정을 전달합니다.
명령은 해당 작업 일정을 삭제합니다.
명령에 Force 매개 *변수가* 포함되어 있기 때문에 확인을 묻는 메시지가 표시되지 않습니다.
