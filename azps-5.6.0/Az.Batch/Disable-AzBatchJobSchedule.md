# Disable-AzBatchJobSchedule

## SYNOPSIS
Batch 작업 일정을 사용하지 않도록 설정합니다.

## 구문

```
Disable-AzBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Disable-AzBatchJobSchedule** cmdlet은 Azure Batch 작업 일정을 사용하지 않도록 설정합니다.
일정을 사용하지 않도록 설정하면 해당 일정에 따라 작업이 생성되지 않습니다.
나중에 비활성화된 일정을 사용하도록 설정할 수 있습니다.

## 예제

### 예제 1: 작업 일정 사용하지 않도록 설정
```
PS C:\>Disable-AzBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

이 명령은 ID JobSchedule17이 있는 작업 일정을 사용하지 않도록 설정합니다.
**Get-AzBatchAccountKey** cmdlet을 사용하여 $Context 할당합니다.
