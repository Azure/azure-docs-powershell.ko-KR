# Get-AzSqlInformationProtectionPolicy

## SYNOPSIS
효과적인 테넌트 SQL 보호 정책을 검색합니다.

## 구문

```
Get-AzSqlInformationProtectionPolicy [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
효과적인 테넌트 SQL 보호 정책을 검색합니다.

## 예제

### 예제
```powershell
PS C:\> Get-AzSqlInformationProtectionPolicy
```
