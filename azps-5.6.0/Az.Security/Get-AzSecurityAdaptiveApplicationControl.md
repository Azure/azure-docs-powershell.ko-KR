# Get-AzSecurityAdaptiveApplicationControl

## SYNOPSIS
구독에 대한 애플리케이션 제어 VM/서버 그룹 목록을 얻습니다.

## 구문

### SubscriptionScope(기본값)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
적응 애플리케이션 컨트롤은 Azure Security Center에서 자동으로 계산되며, 이 cmdlet을 사용하여 구독 범위에서 Adaptive Application Controls 리소스 목록을 얻습니다.
