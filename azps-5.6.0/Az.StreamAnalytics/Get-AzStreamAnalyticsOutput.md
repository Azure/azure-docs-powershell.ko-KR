# Get-AzStreamAnalyticsOutput

## SYNOPSIS
지정된 Stream Analytics 작업 또는 출력에 정의된 출력을 얻습니다.

## 구문

```
Get-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzStreamAnalyticsOutput** cmdlet은 지정된 Stream Analytics 작업에서 정의되거나 특정 출력에 대한 정보를 얻을 수 있는 모든 출력을 나열합니다.

## 예제

### 예제 1: 작업 출력에 대한 정보 얻습니다.
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

이 명령은 StreamingJob 작업에서 정의된 출력에 대한 정보를 반환합니다.

### 예제 2: 특정 작업 출력에 대한 정보 얻기
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

이 명령은 StreamingJob 작업에서 정의된 출력이라는 출력에 대한 정보를 반환합니다.
