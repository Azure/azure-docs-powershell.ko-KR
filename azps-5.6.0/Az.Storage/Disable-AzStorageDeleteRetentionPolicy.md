# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Azure Storage Blob 서비스에 대한 삭제 보존 정책을 사용하지 않도록 설정합니다.

## 구문

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Disable-AzStorageDeleteRetentionPolicy** cmdlet은 Azure Storage Blob 서비스에 대한 삭제 보존 정책을 사용하지 않도록 설정합니다.

## 예제

### 예제 1: Blob 서비스에 대한 삭제 보존 정책을 사용하지 않도록 설정
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

이 명령은 Blob 서비스에 대한 삭제 보존 정책을 사용하지 않도록 설정합니다.
