# Get-AzAppServiceEnvironment

## SYNOPSIS
App Service Environment를 얻습니다. 리소스 그룹만 지정하면 리소스 그룹에서 ASE 목록을 반환합니다.

## 구문

```
Get-AzAppServiceEnvironment [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Get-AzAppServiceEnvironment** cmdlet은 쿼리와 일치하는 ASE를 반환합니다.

## 예제

### 예제 1
```powershell
PS C:\> Get-AzAppServiceEnvironment -ResourceGroupName MyResourceGroup -Name MyAseName
```

에서 명명된 특정 App Service <MyAseName> Environment를 반환합니다. <MyResourceGroup>
