# Get-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Azure Web App에 대한 Access Restiction 구성을 얻습니다.

## 구문

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzWebAppAccessRestrictionConfig** cmdlet은 Azure Web App에 대한 액세스 제한 구성을 얻습니다.

## 예제

### 예제 1: 리소스 그룹에서 웹앱 액세스 제한 구성을 다운로드합니다.
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

이 명령은 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱의 액세스 제한 구성을 얻습니다.
