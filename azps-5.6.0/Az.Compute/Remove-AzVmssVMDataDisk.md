# Remove-AzVmssVMDataDisk

## SYNOPSIS
가상 머신 확장 집합 VM에서 데이터 디스크 제거

## 구문

```
Remove-AzVmssVMDataDisk [-VirtualMachineScaleSetVM] <PSVirtualMachineScaleSetVM> [-Lun] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Remove-AzVmssVMDataDisk** cmdlet은 VM 확장 집합 VM에서 데이터 디스크를 제거합니다.

## 예제

### 예제 1: VM 확장 집합 VM에서 데이터 디스크 제거
```powershell
PS C:\> $VmssVM = Get-AzVmssVM -ResourceGroupName "myrg" -VMScaleSetName "myvmss" -InstanceId 0 
PS C:\> Remove-AzVmssVMDataDisk -VM $VirtualMachine -Lun 0
PS C:\> Update-AzVmssVM -VirtualMachineScaleSetVM $VmssVM
```

첫 번째 명령은 리소스 그룹 이름, vmss 이름 및 인스턴스 ID가 지정한 기존 Vms VM을 얻습니다.
두 번째 명령은 VM에 저장된 VM 확장 집합 VM에서 $VmssVM Vms VM을 제거된 데이터 디스크로 업데이트합니다.
