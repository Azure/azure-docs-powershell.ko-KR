# Get-AzVMImagePublisher

## SYNOPSIS
VMImage 퍼블리셔를 얻습니다.

## 구문

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzVMImagePublisher** cmdlet은 VMImage 퍼블리셔를 얻습니다.

## 예제

### 예제 1: 지역에 대한 VMImage 퍼블리셔를 얻습니다.
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

이 명령은 Azure 프로필 내에서 미국 중부 지역에 대한 VMImage 인스턴스의 게시자를 얻습니다.
