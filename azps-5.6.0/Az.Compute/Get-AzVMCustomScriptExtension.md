# Get-AzVMCustomScriptExtension

## SYNOPSIS
사용자 지정 스크립트 확장에 대한 정보를 얻습니다.

## 구문

```
Get-AzVMCustomScriptExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzVMCustomScriptExtension** cmdlet은 가상 머신의 사용자 지정 스크립트 Virtual Machine 확장에 대한 정보를 제공합니다.

## 예제

### 예제 1: 사용자 지정 스크립트 확장 사용
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript"
```

이 명령은 VirtualMachine07이라는 가상 머신에 대해 ContosoCustomScript라는 사용자 지정 스크립트 확장을 얻습니다.

### 예제 2: 사용자 지정 스크립트 확장의 인스턴스 보기를 얻게 됩니다.
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript" -Status
```

이 명령은 VirtualMachine07이라는 가상 머신에 대해 ContosoCustomScript라는 사용자 지정 스크립트 확장의 인스턴스 보기를 얻습니다.
