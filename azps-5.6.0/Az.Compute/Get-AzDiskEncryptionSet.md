# Get-AzDiskEncryptionSet

## SYNOPSIS
디스크 암호화 집합을 얻거나 나열합니다.

## 구문

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
디스크 암호화 집합을 얻거나 나열합니다.

## 예제
