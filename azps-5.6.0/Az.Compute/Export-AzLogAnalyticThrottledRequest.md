# Export-AzLogAnalyticThrottledRequest

## SYNOPSIS
주어진 시간 창에서 이 구독에 대한 총 스로틀된 Api 요청을 표시하는 로그를 내보낼 수 있습니다.

## 구문

```
Export-AzLogAnalyticThrottledRequest [-Location] <String> [-FromTime] <DateTime> [-ToTime] <DateTime>
 [-BlobContainerSasUri] <String> [-GroupByOperationName] [-GroupByResourceName] [-GroupByThrottlePolicy]
 [-GroupByApplicationId] [-GroupByUserAgent] [-AsJob] [-NoWait] [-DefaultProfile <IAzureContextContainer>] 
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
이렇게 하여 Microsoft.Compute API 호출의 총 수를 내보낼 수 있습니다.
로그는 GroupByOperationName, GroupByThrottlePolicy, GroupByResourceName, GroupByUserAgent 또는 GroupByApplicationId의 5개 옵션으로 추가로 집계할 수 있습니다.
이 cmdlet은 CRP 로그만 수집합니다.

컴퓨팅 리소스 공급자의 API 제한에 대한 개요는 https://docs.microsoft.com/azure/azure-resource-manager/resource-manager-request-limits 를 참조하세요. 

## 예제

### 예제 1: 작업 이름으로 집계된 레코드 내보내기
```
PS C:\> Export-AzLogAnalyticThrottledRequest -Location 'West Central US' -FromTime '2018-02-20T17:54:14.8806951-08:00' -ToTime '2018-02-22T17:54:17.5832413-08:00' -BlobContainerSasUri 'https://wkuotest1.blob.core.windows.net/mylogs?someSasUri' -GroupByOperationName
```

이 명령은 2018-02-20T17:54:14와 2018-02-22T17:54:17 사이의 총 Microsoft.Compute API 호출을 작업 이름으로 집계합니다.

### 예제 2: 애플리케이션 ID로 집계된 레코드 내보내기
```
PS C:\> Export-AzLogAnalyticThrottledRequest -Location 'West Central US' -FromTime '2018-02-20T17:54:14.8806951-08:00' -ToTime '2018-02-22T17:54:17.5832413-08:00' -BlobContainerSasUri 'https://wkuotest1.blob.core.windows.net/mylogs?someSasUri' -GroupByApplicationId
```

이 명령은 2018-02-20T17:54:14와 2018-02-22T17:54:17 사이의 총 Microsoft.Compute API 호출을 적용 ID로 집계합니다.

### 예제 3: 사용자 에이전트에 의해 집계된 레코드 내보내기
```
PS C:\> Export-AzLogAnalyticThrottledRequest -Location 'West Central US' -FromTime '2018-02-20T17:54:14.8806951-08:00' -ToTime '2018-02-22T17:54:17.5832413-08:00' -BlobContainerSasUri 'https://wkuotest1.blob.core.windows.net/mylogs?someSasUri' -GroupByUserAgent
```

이 명령은 사용자 에이전트에 의해 집계된 주어진 SAS URI에 2018-02-20T17:54:14 및 2018-02-22T17:54:17 사이의 총 스로틀된 Microsoft.Compute API 호출을 저장합니다.
