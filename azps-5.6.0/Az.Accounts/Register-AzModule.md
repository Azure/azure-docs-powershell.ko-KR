# Register-AzModule

## SYNOPSIS
내부 사용 전용 - AutoRest 생성된 cmdlet에 대한 런타임 지원 제공

## 구문

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
내부 사용 전용 - AutoRest 생성된 cmdlet에 대한 런타임 지원 제공

## 예제

### 예제 1
```powershell
PS C:\> Register-AzModule
```
