# Get-AzHpcCacheSku

## SYNOPSIS
구독에서 사용할 수 있는 모든 SKUS를 얻습니다.

## 구문

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzHpcCacheSku** cmdlet은 구독에서 사용할 수 있는 SKU 목록을 반환합니다.

## 예제

### 예제 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
