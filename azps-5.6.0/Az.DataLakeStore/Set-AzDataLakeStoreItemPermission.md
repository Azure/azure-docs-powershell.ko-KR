# Set-AzDataLakeStoreItemPermission

## SYNOPSIS
Data Lake Store에서 파일 또는 폴더의 사용 권한 octal을 수정합니다.

## 구문

```
Set-AzDataLakeStoreItemPermission [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Permission] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Set-AzDataLakeStoreItemPermission** cmdlet은 Data Lake Store의 파일 또는 폴더의 사용 권한 octal을 수정합니다.

## 예제

### 예제 1: 항목에 대한 사용 권한 octal 설정
```
PS C:\>Set-AzDataLakeStoreItemPermission -AccountName "ContosoADL" -Path "/file.txt" -Permission 0770
```

이 명령은 파일의 사용 권한 octal을 0770으로 설정하여 고정 비트를 지우고, 파일의 소유자에 대한 읽기/쓰기/실행 사용 권한을 설정하고, 파일의 소유 그룹에 대한 읽기/쓰기/실행 사용 권한을 설정하고, 읽기/쓰기/실행 권한을 지우는 것으로 변환합니다.
