# Az.Subscription 모듈
## 설명
이 섹션의 항목에서는 Azure Resource Manager(Azure Resource Manager) 프레임워크의 Azure 구독에 대한 Azure PowerShell cmdlet을 ARM 설명합니다. cmdlet은 Microsoft.Azure.Commands.Subscription 네임스페이스에 있습니다.

## Az.Subscription Cmdlet
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
구독 별칭 세부 정보 얻습니다.

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
새 별칭 및 구독을 만듭니다.

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
구독 별칭 삭제

### [Update-AzSubscription](Update-AzSubscription.md)
Azure 구독 업데이트
