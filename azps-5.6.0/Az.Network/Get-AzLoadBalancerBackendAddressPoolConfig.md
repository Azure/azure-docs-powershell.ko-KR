# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
부하 저울에 대한 백던드 주소 풀 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerBackendAddressPoolConfig** cmdlet은 단일 백end 주소 풀 또는 부하 분산기 내의 백end 주소 풀 목록을 제공합니다.

## 예제

### 예제 1: 백던드 주소 풀을 얻다
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

첫 번째 명령은 MyResourceGroup이라는 리소스 그룹에 MyLoadBalancer라는 기존 부하 분산 장치 $loadbalancer 합니다.
두 번째 명령은 2018년의 부하 균형 조정기에 대해 BackendAddressPool02라는 연결된 백end 주소 풀 구성을 $loadbalancer.
