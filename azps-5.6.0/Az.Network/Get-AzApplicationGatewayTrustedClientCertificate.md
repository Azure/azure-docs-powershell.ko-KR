# Get-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Application Gateway에서 특정 이름으로 신뢰할 수 있는 클라이언트 CA 인증서 체인을 얻습니다.

## 구문

```
Get-AzApplicationGatewayTrustedClientCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
Get-AzApplicationGatewayTrustedClientCertificate cmdlet은 Application Gateway에서 특정 이름으로 신뢰할 수 있는 클라이언트 CA 인증서 체인을 얻습니다.

## 예제

### 예제 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClientCert = Get-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName
```

첫 번째 명령은 Application Gateway를 얻게 하여 변수에 $gw 저장합니다. 두 번째 명령은 Application Gateway에서 지정된 이름으로 신뢰할 수 있는 클라이언트 CA 인증서 체인을 얻습니다.
