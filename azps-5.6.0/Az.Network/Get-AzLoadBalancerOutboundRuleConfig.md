# Get-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
부하 저울에서 아웃바운드 규칙 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerOutboundRuleConfig** cmdlet은 부하 분산기에서 아웃바운드 규칙 구성 또는 아웃바운드 규칙 구성 목록을 제공합니다.

## 예제

### 예제 1: 부하 저울에서 아웃바운드 규칙 구성을 얻다
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer $slb -Name "MyRule"
```

첫 번째 명령은 MyLoadBalancer라는 부하 분산 변수를 얻은 다음, 변수에 $slb.
두 번째 명령은 해당 부하 균형 조정기와 연결된 MyRule이라는 아웃바운드 규칙 구성을 얻습니다.
