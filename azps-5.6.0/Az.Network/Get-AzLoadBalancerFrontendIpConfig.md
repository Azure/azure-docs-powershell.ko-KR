# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
부하 균형 조정기에서 프런트 엔드 IP 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerFrontendIpConfig** cmdlet은 프런트 엔드 IP 구성 또는 부하 분산기에서 프런트 엔드 IP 구성 목록을 제공합니다.

## 예제

### 예제 1: 부하 균형 조정기에서 프런트 엔드 IP 구성 얻기
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

첫 번째 명령은 MyLoadBalancer라는 부하 분산 변수를 얻은 다음, 변수에 $slb.
두 번째 명령은 해당 부하 균형 조정기와 연결된 프런트 엔드 IP 구성을 얻습니다.
