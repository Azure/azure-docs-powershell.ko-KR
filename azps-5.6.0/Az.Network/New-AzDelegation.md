# New-AzDelegation

## SYNOPSIS
서비스 위임을 만듭니다.

## 구문

```
New-AzDelegation -Name <String> -ServiceName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**New-AzDelegation** cmdlet은 서브넷에 추가할 수 있는 서비스 위임을 만듭니다.

## 예제

### 예제 1
```powershell
PS C:\> $delegation = New-AzDelegation -Name "myDelegation" -ServiceName "Microsoft.Sql/servers"
PS C:\> $vnet = Get-AzVirtualNetwork -Name "myVNet" -ResourceGroupName "myResourceGroup"
PS C:\> $subnet = Get-AzVirtualNetworkSubnetConfig -Name "mySubnet" -VirtualNetwork $vnet
PS C:\> $subnet.Delegations.Add($delegation)
PS C:\> Set-AzVirtualNetwork $vnet
```

첫 번째 cmdlet은 서브넷에 추가할 수 있는 위임을 만들고 이 위임은 $delegation 변수에 저장합니다. 두 번째 및 세 번째 cmdlet은 위임할 서브넷을 검색합니다. 네 번째 cmdlet은 만든 위임이 관심 있는 서브넷에 추가하고 최종 cmdlet은 업데이트된 구성을 서버에 전송합니다.
