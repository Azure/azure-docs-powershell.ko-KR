# Get-AzLoadBalancerRuleConfig

## SYNOPSIS
부하 저울에 대한 규칙 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerRuleConfig** cmdlet은 부하 분산기에 대한 하나 이상의 규칙 구성을 제공합니다.

## 예제

### 예제 1: 부하 저울의 규칙 구성을 얻게 됩니다.
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerRuleConfig -Name "MyLBrulename" -LoadBalancer $slb
```

첫 번째 명령은 MyLoadBalancer라는 부하 분산 변수를 얻은 다음, 변수에 $slb.
두 번째 명령은 MyLBrulename이라는 연결된 규칙 구성을 $slb.
