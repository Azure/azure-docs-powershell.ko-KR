# Get-AzNetworkVirtualApplianceSku

## SYNOPSIS
인벤토리에서 사용 가능한 네트워크 가상 어플라이언스 Sku를 얻거나 나열합니다.

## 구문

```
Get-AzNetworkVirtualApplianceSku [-SkuName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
Microsoft Get-AzNetworkVirtualApplianceSku 인벤토리에서 사용 가능한 네트워크 가상 어플라이언스 Sku를 얻거나 나열합니다.
