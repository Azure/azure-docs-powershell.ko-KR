# Get-AzLoadBalancerProbeConfig

## SYNOPSIS
부하 균형 조정기에 대한 프로브 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerProbeConfig** cmdlet은 부하 분산기에 대한 하나 이상의 프로브 구성을 제공합니다.

## 예제

### 예제 1: 부하 저울의 프로브 구성을 얻게 됩니다.
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $slb
```

첫 번째 명령은 MyLoadBalancer라는 부하 분산 변수를 얻은 다음, 변수에 $slb.
두 번째 명령은 두 번째 명령의 부하 균형기에서 MyProbe라는 연결된 프로브 구성을 $slb.
