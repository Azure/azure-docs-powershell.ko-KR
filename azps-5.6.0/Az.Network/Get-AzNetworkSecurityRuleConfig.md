# Get-AzNetworkSecurityRuleConfig

## SYNOPSIS
네트워크 보안 그룹에 대한 네트워크 보안 규칙 구성을 확인합니다.

## 구문

```
Get-AzNetworkSecurityRuleConfig [-Name <String>] -NetworkSecurityGroup <PSNetworkSecurityGroup> [-DefaultRules]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzNetworkSecurityRuleConfig** cmdlet은 Azure 네트워크 보안 그룹에 대한 네트워크 보안 규칙 구성을 제공합니다.

## 예제

### 1: 네트워크 보안 규칙 구성 검색
```
Get-AzNetworkSecurityGroup -Name nsg1 -ResourceGroupName rg1 
    | Get-AzNetworkSecurityRuleConfig -Name AllowInternetOutBound -DefaultRules
```

이 명령은 리소스 그룹 "rg1"에서 "nsg1"이라는 Azure 네트워크 보안 그룹에서 "AllowInternetOutBound"이라는 기본 규칙을 검색합니다.

### 2: 이름만 사용하여 네트워크 보안 규칙 구성 검색
```
Get-AzNetworkSecurityGroup -Name nsg1 -ResourceGroupName rg1 
    | Get-AzNetworkSecurityRuleConfig -Name "rdp-rule"
```

이 명령은 리소스 그룹 "rg1"에서 "nsg1"이라는 Azure 네트워크 보안 그룹에서 "rdp-rule"이라는 사용자 정의 규칙을 검색합니다.
