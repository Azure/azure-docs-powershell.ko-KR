# Remove-AzApplicationGatewayIdentity

## SYNOPSIS
애플리케이션 게이트웨이에서 ID를 제거합니다.

## 구문

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzApplicationGatewayIdentity** cmdlet은 애플리케이션 게이트웨이에서 ID를 제거합니다.

## 예제

### 예제 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

이 예제에서는 기존 애플리케이션 게이트웨이에서 ID를 제거합니다.
참고: 게이트웨이가 keyvault 비밀을 참조하는 경우 이 작업을 따라 해당 ssl 인증서 참조를 제거하는 것이 중요합니다.
