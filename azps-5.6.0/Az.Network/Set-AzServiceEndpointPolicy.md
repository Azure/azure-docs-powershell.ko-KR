# Set-AzServiceEndpointPolicy

## SYNOPSIS
서비스 엔드포인트 정책을 업데이트합니다.

## 구문

```
Set-AzServiceEndpointPolicy -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Set-AzServiceEndpointPolicy** cmdlet은 서비스 엔드포인트 정책을 생성합니다.

## 예제

### 예제 1: 서비스 엔드포인트 정책 설정
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicy -Name "Policy1" -ServiceEndpointPolicy $serviceEndpointPolicy -ResourceGroup "resourcegroup1"
```

이 명령은 개체가 정의한 Policy1이라는 서비스 엔드포인트 정책을 $serviceEndpointPolicy 리소스 그룹 "resourcegroup1"에 속합니다.
