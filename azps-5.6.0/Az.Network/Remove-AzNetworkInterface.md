# Remove-AzNetworkInterface

## SYNOPSIS
네트워크 인터페이스를 제거합니다.

## 구문

```
Remove-AzNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzNetworkInterface** cmdlet은 Azure 네트워크 인터페이스를 제거합니다.

## 예제

### 예제 1: 네트워크 인터페이스 제거
```
PS C:\>Remove-AzNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

이 명령은 리소스 그룹 ResourceGroup1에서 NetworkInterface1 네트워크 인터페이스를 제거합니다.
Force *매개* 변수를 사용할 수 없습니다. 사용자에게 이 작업을 확인하라는 메시지가 표시됩니다.

### 예제 2: 네트워크 인터페이스 제거
```
PS C:\>Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzNetworkInterface -Force
```

이 명령은 ResourceGroup1 리소스 그룹의 모든 네트워크 인터페이스를 제거합니다.
Force *매개* 변수가 사용되어 있기 때문에 사용자는 확인을 묻는 메시지가 없습니다.
