# Get-AzRouteFilterRuleConfig

## SYNOPSIS
경로 필터에서 경로 필터 규칙을 얻습니다.

## 구문

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzRouteFilterRuleConfig** cmdlet은 경로 필터 규칙 또는 경로 필터의 경로 필터 규칙 목록을 얻습니다.

## 예제

### 예제 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

첫 번째 명령은 MyRouteFilter라는 경로 필터를 얻은 다음, 변수에 $rf.
두 번째 명령은 해당 경로 필터와 연결된 Rule01이라는 경로 필터 규칙을 얻습니다.
세 번째 명령은 해당 경로 필터와 연결된 경로 필터 규칙 목록을 얻습니다.
