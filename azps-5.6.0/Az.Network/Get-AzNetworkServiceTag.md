# Get-AzNetworkServiceTag

## SYNOPSIS
서비스 태그 정보 리소스 목록을 얻습니다.

## 구문

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzNetworkServiceTag** cmdlet은 서비스 태그 정보 리소스 목록을 얻습니다.

지정한 Azure 지역 정보는 위치에 따라 필터가 아닌 버전에 대한 참조로 사용됩니다. 예를 들어 지정한 경우에도 구독이 속한 클라우드(예: 공용, 미국 정부, 중국 또는 독일)에 속하는 클라우드로 제한되는 모든 지역에 대한 세부 정보가 있는 서비스 태그 목록을 얻을 수 `-Location eastus2` 있습니다.

두 번째 명령은 목록을 필터하여 Sql에 대한 정보 리소스를 선택합니다.

### 예제 3: 미국 서부 2에 대한 Storage의 서비스 태그 정보 리소스 얻기
```powershell

Region           : westus2
Address Prefixes : {13.66.176.16/28, 13.66.176.48/28, 13.66.232.64/28, 13.66.232.208/28...}
```

첫 번째 명령은 서비스 태그 정보 리소스 목록을 얻게 하여 변수에 `serviceTags` 저장합니다.
다음 명령은 목록을 필터링하여 미국 서부 2의 Storage에 대한 서비스 태그 정보를 선택하는 다양한 방법을 보여줄 수 있습니다.
