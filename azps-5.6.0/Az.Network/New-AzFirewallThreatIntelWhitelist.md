# New-AzFirewallThreatIntelWhitelist

## SYNOPSIS
Azure Firewall에 대한 새 위협 인텔리전스 허용 목록 만들기

## 구문

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**New-AzFirewallThreatIntelWhitelist** cmdlet은 Azure 방화벽을 만들거나 설정할 때 사용할 수 있는 위협 인텔 허용 목록 개체를 만듭니다.

## 예제

### 예제 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

이 예제에서는 두 항목의 FQDN 화이트리스트와 두 항목의 IP 주소 화이트리스트가 포함된 위협 인텔 화이트리스트를 만듭니다.
