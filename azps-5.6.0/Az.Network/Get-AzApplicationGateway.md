# Get-AzApplicationGateway

## SYNOPSIS
애플리케이션 게이트웨이를 얻습니다.

## 구문

```
Get-AzApplicationGateway [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzApplicationGateway** cmdlet은 애플리케이션 게이트웨이를 얻습니다.
