# Add-AzDelegation

## SYNOPSIS
서브넷에 위임이 추가됩니다.

## 구문

```
Add-AzDelegation -Name <String> -ServiceName <String> -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Add-AzDelegation** cmdlet은 Azure 서브넷에 서비스 위임이 추가됩니다.

## 예제

### 예제 1: 위임 추가
```powershell
PS C:\> $vnet = Get-AzVirtualNetwork -Name "myVNet" -ResourceGroupName "myResourceGroup"
PS C:\> $subnet = Get-AzVirtualNetworkSubnetConfig -Name "mySubnet" -VirtualNetwork $vnet
PS C:\> $subnet = Add-AzDelegation -Name "myDelegation" -ServiceName "Microsoft.Sql/servers" -Subnet $subnet
PS C:\> Set-AzVirtualNetwork $vnet
```

첫 번째 명령은 서브넷이 있는 가상 네트워크를 검색합니다. 두 번째 명령은 관심 있는 서브넷(위임할 서브넷)을 격리합니다. 세 번째 명령은 서브넷에 위임이 추가됩니다. 이 특정 예제는 이 서브넷에서 인터페이스 엔드포인트를 SQL 설정하려는 경우 유용합니다. 최종 명령은 업데이트된 서브넷을 서버에 보내서 서브넷을 실제로 업데이트합니다.
