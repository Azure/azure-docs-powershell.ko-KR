# Get-AzDdosProtectionPlan

## SYNOPSIS
DDoS 보호 계획을 얻습니다.

## 구문

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
Get-AzDdosProtectionPlan cmdlet은 DDoS 보호 계획을 얻습니다.

## 예제

### 예제 1: 특정 DDoS 보호 계획 사용
```

                      {

                      }

```

이 경우 리소스 그룹 및 DDoS 보호 계획의 이름에 해당하는 **ResourceGroupName** 및 **Name** 특성을 각각 지정해야 합니다.
