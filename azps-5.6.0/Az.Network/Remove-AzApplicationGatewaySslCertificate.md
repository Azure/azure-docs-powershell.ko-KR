# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Azure 애플리케이션 게이트웨이에서 SSL 인증서를 제거합니다.

## 구문

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Remove-AzApplicationGatewaySslCertificate** cmdlet은 Azure 애플리케이션 게이트웨이에서 SSL(Secure Sockets Layer) 인증서를 제거합니다.

## 예제

### 예제 1: 애플리케이션 게이트웨이에서 SSL 인증서 제거
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

첫 번째 명령은 ApplicationGateway01이라는 애플리케이션 게이트웨이를 얻게 하여 결과를 $AppGW.
두 번째 명령은 애플리케이션 게이트웨이에서 Cert02라는 SSL 인증서를 $AppGW 제거합니다.
