# Get-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
부하 저울에 대한 인바운드 NAT 규칙 구성을 얻습니다.

## 구문

```
Get-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzLoadBalancerInboundNatRuleConfig** cmdlet은 Azure 부하 분산기에서 하나 이상의 NAT(인바운드 네트워크 주소 변환) 규칙을 얻습니다.

## 예제

### 예제 1: 인바운드 NAT 규칙 구성을 얻다
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerInboundNatRuleConfig -Name "MyInboundNatRule1" -LoadBalancer $slb
```

첫 번째 명령은 MyLoadBalancer라는 부하 분산 변수를 얻게 하여 변수에 $slb.
두 번째 명령은 MyInboundNatRule1이라는 연결된 NAT 규칙을 부하 $slb.
