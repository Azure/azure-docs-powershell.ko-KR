# Get-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
백던드 서버 풀에 대한 URL 경로 매핑 배열을 얻습니다.

## 구문

```
Get-AzApplicationGatewayUrlPathMapConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzApplicationGatewayURLPathMapConfig** cmdlet은 백END 서버 풀에 대한 URL 경로 매핑 배열을 제공합니다.

## 예제

### 예제 1: URL 경로 맵 구성을 다운로드합니다.
```
PS C:\>Get-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway Gateway
```

이 명령은 게이트웨이라는 애플리케이션 게이트웨이에 있는 백end 서버에서 URL 경로 맵 구성을 얻습니다.
