# Get-AzRecoveryServicesAsrProtectionContainer

## SYNOPSIS
Recovery Services 자격 증명 모음에서 ASR 보호 컨테이너를 얻습니다.

## 구문

### ByFabricObject(기본값)
```
Get-AzRecoveryServicesAsrProtectionContainer -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByObjectWithName
```
Get-AzRecoveryServicesAsrProtectionContainer -Name <String> -Fabric <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectWithFriendlyName
```
Get-AzRecoveryServicesAsrProtectionContainer -FriendlyName <String> -Fabric <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzRecoveryServicesAsrProtectionContainer** cmdlet은 Recovery Services 자격 증명 모음에서 Azure Site Recovery 보호 컨테이너를 얻습니다.
보호 컨테이너는 보호 가능(검색) 및 보호된 개체(예: 가상 머신)에 대한 논리 컨테이너입니다.
복제 정책은 보호된 항목에 대한 복제 설정을 정의하며 보호 컨테이너와 연결하여 보호 가능한 항목에 적용할 수 있습니다.
