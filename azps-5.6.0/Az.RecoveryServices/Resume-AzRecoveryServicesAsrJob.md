# Resume-AzRecoveryServicesAsrJob

## SYNOPSIS
일시 중단된 Azure Site Recovery 작업을 다시 시작합니다.

## 구문

### ByObject(기본값)
```
Resume-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-Comment <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Resume-AzRecoveryServicesAsrJob -Name <String> [-Comment <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Resume-AzRecoveryServicesAsrJob** cmdlet은 일시 중단된 Azure Site Recovery 작업을 다시 시작합니다.

## 예제

### 예제 1
```
PS C:\> $currentJob = Resume-AzRecoveryServicesAsrJob -Job $Job
```

대기 중 또는 일시 중단 상태인 경우 지정된 작업을 다시 시작하고 ASR 작업에 해당하는 업데이트된 ASR 작업 개체를 반환합니다.
