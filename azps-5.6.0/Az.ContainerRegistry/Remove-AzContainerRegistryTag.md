# Remove-AzContainerRegistryTag

## SYNOPSIS
ACR 태그를 언태그합니다.

## 구문

```
Remove-AzContainerRegistryTag -RepositoryName <String> -Name <String> -RegistryName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
ACR 태그를 언태그합니다.
이 cmdlet을 사용 하시기 바랍니다. `Update-AzContainerRegistryRepository -RegistryName XXX -Repository XXX -DeleteEnable $true`
먼저.

## 예제

### 예제 1
```powershell
Remove-AzContainerRegistryTag -RegistryName registry -RepositoryName alpine -Name latest
True
```

레지스트리 아래에서 alpine:tag 태그를 태그 지정합니다.
