# Az.ManagedServiceIdentity 모듈
## 설명
이 항목에서는 Azure Managed Service Identity Cmdlet에 대한 도움말 토픽을 제공합니다.

## Az.ManagedServiceIdentity Cmdlet
### [Get-AzUserAssignedIdentity](Get-AzUserAssignedIdentity.md)
사용자 할당 ID/ID를 얻습니다.

### [New-AzUserAssignedIdentity](New-AzUserAssignedIdentity.md)
새 사용자 할당 ID를 생성하거나 기존 사용자 할당 ID를 업데이트합니다.

### [Remove-AzUserAssignedIdentity](Remove-AzUserAssignedIdentity.md)
사용자 할당 ID를 제거합니다.
