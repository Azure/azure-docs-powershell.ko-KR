# Get-AzAutomationCertificate

## SYNOPSIS
Automation 인증서를 얻습니다.

## 구문

### ByAll(기본값)
```
Get-AzAutomationCertificate [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByCertificateName
```
Get-AzAutomationCertificate [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzAutomationCertificate** cmdlet은 하나 이상의 Azure Automation 인증서를 얻습니다.
기본적으로 이 cmdlet은 모든 인증서를 얻습니다.
특정 인증서를 얻을 인증서의 이름을 지정합니다.

## 예제

### 예제 1: 모든 인증서를 얻습니다.
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17"
```

이 명령은 Contoso17이라는 Automation 계정의 모든 인증서에 대한 메타데이터를 얻습니다.

### 예제 2: 인증서를 얻다
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17" -Name "ContosoCertificate"
```

이 명령은 ContosoCertificate라는 인증서에 대한 메타데이터를 얻습니다.
