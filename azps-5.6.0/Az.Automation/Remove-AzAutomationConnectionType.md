# Remove-AzAutomationConnectionType

## SYNOPSIS
Automation 연결 형식을 제거합니다.

## 구문

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Remove-AzAutomationConnectionType** cmdlet은 Azure Automation에서 연결 유형을 제거합니다.
삭제한 연결 유형과 연결된 모든 연결은 사용할 수 없습니다.
다음 조건을 충족하는 새 연결 유형을 만들지 않는 한 제거합니다. 
- 형식에는 원래 연결 형식과 이름이 동일합니다. 
- 형식에는 원래 연결 유형과 동일한 필드 정의가 있습니다.
추가 필드를 사용할 수 있습니다.

## 예제

### 예제 1: 연결 유형 제거
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

이 명령은 Contoso17이라는 Automation 계정에서 ContosoConnectionType이라는 연결 유형을 제거합니다.
