# Set-AzApiManagementGroup

## SYNOPSIS
API 관리 그룹을 구성합니다.

## 구문

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Set-AzApiManagementGroup** cmdlet은 API 관리 그룹을 구성합니다.

## 예제

### 예제 1: 관리 그룹 구성
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -GroupId "0001" -Description "Updated Management Group" -Name "Group0001"
```

이 명령은 Group0001이라는 관리 그룹을 구성합니다.
