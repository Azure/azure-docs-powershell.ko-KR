# Remove-AzApiManagementApiRevision

## SYNOPSIS
특정 API 개정을 제거했습니다.

## 구문

### ByApiId(기본값)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
cmdlet **Remove-AzApiManagementApiRevision은** 특정 API 개정을 제거합니다.

## 예제

### 예제 1: API 개정 제거
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

이 명령은 API Management 서비스에서 `2` API의 `echo-api` 개정을 제거합니다.

### 예제 2

이 명령은 API Management 서비스에서 API echo-api의 2개 개정을 제거합니다. (자동 재생)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
