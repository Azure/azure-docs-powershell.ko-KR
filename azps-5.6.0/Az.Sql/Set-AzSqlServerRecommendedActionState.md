# Set-AzSqlServerRecommendedActionState

## SYNOPSIS
권장되는 작업에서 Azure SQL Server 업데이트합니다.

## 구문

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Set-AzSqlServerRecommendedActionState** cmdlet은 Azure 권장 작업의 SQL Server 업데이트합니다.
이 cmdlet은 새 상태를 기반으로 권장되는 작업을 적용, 되버트 또는 삭제합니다.
