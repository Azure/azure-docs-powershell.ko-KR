# Get-AzSqlDatabaseDataMaskingPolicy

## SYNOPSIS
데이터베이스에 대한 데이터 마스킹 정책을 얻습니다.

## 구문

```
Get-AzSqlDatabaseDataMaskingPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Get-AzSqlDatabaseDataMaskingPolicy** cmdlet은 Azure SQL 마스킹 정책을 제공합니다.
이 cmdlet을 사용하려면 *ResourceGroupName,* *ServerName* 및 *DatabaseName* 매개 변수를 사용하여 데이터베이스를 식별합니다.
이 cmdlet은 Azure의 SQL Server Stretch Database 서비스에서 지원됩니다.

## 예제

### 예제 1: Azure 데이터베이스의 데이터 마스킹 정책 SQL 데이터 마스킹
```
PS C:\>Get-AzSqlDatabaseDataMaskingPolicy -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
DatabaseName      : Database01
ResourceGroupName : ResourceGroup01
ServerName        : Server01
DataMaskingState  : Enabled
PrivilegedUsers  :
```

이 명령은 server Server01의 리소스 그룹 ResourceGroup01의 Database Database01에서 데이터 마스킹 정책을 얻습니다.
