# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
두 서버 간의 탄력적 데이터베이스 트랜잭션에 대한 통신 링크를 삭제합니다.

## 구문

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Remove-AzSqlServerCommunicationLink** cmdlet은 Azure SQL 데이터베이스에서 탄력적 데이터베이스 트랜잭션에 대한 서버 간 통신 링크를 삭제합니다.

## 예제

### 예제 1: 통신 링크 삭제
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

이 명령은 ContosoServer17의 Link01이라는 서버 간 통신 링크를 삭제합니다.
