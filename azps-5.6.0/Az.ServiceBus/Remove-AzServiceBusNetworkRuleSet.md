# Remove-AzServiceBusNetworkRuleSet

## SYNOPSIS
주어진 네임스페이스에 대한 NetworkRuleSet을 제거합니다.

## 구문

### NetworkRuleSetPropertiesSet(기본값)
```
Remove-AzServiceBusNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NamespaceInputObjectSet
```
Remove-AzServiceBusNetworkRuleSet [-InputObject] <PSNamespaceAttributes> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NetworkRuleSetResourceIdParameterSet
```
Remove-AzServiceBusNetworkRuleSet [-ResourceId] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
주어진 네임스페이스에 대한 NetworkRuleSet을 제거합니다.

## 예제

### 예제 1
```powershell
PS C:\> Remove-AzServiceBusNetworkRuleSet -ResourceGroupName  v-ajnavtest -Namespace ServiceBus-Namespace1-1375
```

이름 : defaultAction : Id 허용 : /subscriptions/subscriptionId/resourceGroups/ResourceGroup/providers/Microsoft.ServiceBus/namespaces/serviceBus-Namespace-1375/networkRuleSets/default Type : Microsoft.ServiceBus/Namespaces/NetworkRuleSet IpRules : VirtualNetworkRules : 

주어진 "ServiceBus-Namespace1-1375" 네임스페이스에 대한 NetworkRuleSet을 삭제합니다. 

### 예제 2
```powershell
PS C:\> Remove-AzServiceBusNetworkRuleSet -InputObject $result1375
```
이름 : defaultAction : Id 허용 : /subscriptions/subscriptionId/resourceGroups/RSG-TestAzEventhub/providers/Microsoft.EventHub/namespaces/Eventhub-Namespace-1375/networkRuleSets/default Type : Microsoft.EventHub/namespaces/NetworkRuleSet IpRules : VirtualNetworkRules : 

InputObject를 사용하여 NetworkRuleSet을 삭제합니다. 

### 예제 3
```powershell
PS C:\> Remove-AzServiceBusNetworkRuleSet -ResourceId /SubscriptionId/resourcegroups/ResourceGroup/providers/Microsoft.ServiceBus/namespaces/ServiceBus-Namespace1-1375
```
이름 : defaultAction : Id 허용 : /subscriptions/subscriptionId/resourceGroups/RSG-TestAzEventhub/providers/Microsoft.EventHub/namespaces/Eventhub-Namespace-1375/networkRuleSets/default Type : Microsoft.EventHub/namespaces/NetworkRuleSet IpRules : VirtualNetworkRules : 

네임스페이스의 ResourceId를 사용하여 NetworkRuleSet을 삭제합니다.
