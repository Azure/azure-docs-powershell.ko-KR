# <a name="overview-of-azure-deployment-manager"></a>Azure Deployment Manager 개요

Deployment Manager를 사용하면 여러 지역에 걸쳐 서비스를 배포할 때 안전한 배포 사례를 사용할 수 있습니다.

Azure Deployment Manager가 공개 미리 보기로 제공됩니다. 미리보기에 등록하고 Azure Deployment Manager를 사용하는 방법을 배우려면 [Azure Deployment Manager를 사용하는 안전한 배포 방법 사용](/azure/azure-resource-manager/deployment-manager-overview)을 참조하세요. [Resource Manager 템플릿을 사용하여 Azure Deployment Manager 사용하기](/azure/azure-resource-manager/deployment-manager-tutorial) 자습서를 참조하세요.
