# New-AzAutomationKey

## SYNOPSIS
Automation 계정에 대한 등록 키를 다시 생성합니다.

## 구문

```
New-AzAutomationKey [-KeyType] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**New-AzAutomationKey** cmdlet은 Azure Automation 계정에 대한 등록 키를 다시 생성합니다.

## 예제

### 예제 1: Automation 계정에 대한 키 다시 생성
```
PS C:\>New-AzAutomationKey -KeyType Primary -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

이 명령은 ResourceGroup01이라는 리소스 그룹에 AutomationAccount01이라는 Azure Automation 계정에 대한 기본 키를 다시 생성합니다.
