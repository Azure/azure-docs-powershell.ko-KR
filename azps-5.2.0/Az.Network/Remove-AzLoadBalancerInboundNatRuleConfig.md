# Remove-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
부하 균형 조정기에서 인바운드 NAT 규칙 구성을 제거합니다.

## 구문

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzLoadBalancerInboundNatRuleConfig** cmdlet은 Azure Load Balancer에서 인바운드 NAT(네트워크 주소 변환) 규칙 구성을 제거합니다.

## 예제

### 1: Azure Load Balancer에서 인바운드 NAT 규칙 삭제
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

첫 번째 명령은 "mylb"라는 기존 부하를 로드하고 부하 $load 저장합니다. 두 번째 명령은 이 부하 균형 조정과 연결된 인바운드 NAT 규칙을 제거합니다.
