# Remove-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
애플리케이션 게이트웨이에서 백 엔드 HTTP 설정을 제거합니다.

## 구문

```
Remove-AzApplicationGatewayBackendHttpSetting -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
이 Remove-AzApplicationGatewayBackendHttpSetting cmdlet은 Azure 애플리케이션 게이트웨이에서 백 엔드 HTTP(Hypertext Transfer Protocol) 설정을 제거합니다.

## 예제

### 예제 1: 애플리케이션 게이트웨이에서 백 엔드 HTTP 설정 제거
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw -Name "BackEndSetting02"
```

첫 번째 명령은 ResourceGroup01이라는 리소스 그룹에 속하는 ApplicationGateway01이라는 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다.
두 번째 명령은 백 엔드에 저장된 애플리케이션 게이트웨이에서 BackEndSetting02라는 백 엔드 HTTP 설정을 $AppGw.
