# Get-AzDdosProtectionPlan

## SYNOPSIS
DDoS 보호 계획을 얻습니다.

## 구문

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
Get-AzDdosProtectionPlan cmdlet은 DDoS 보호 계획을 얻습니다.

## 예제

### 예제 1: 특정 DDoS 보호 계획 얻기
```
D:\> Get-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName

                      {
                        "Id": "/subscriptions/d1dbd366-9871-45ac-84b7-fb318152a9e0/resourceGroups/ResourceGroupName/providers/Microsoft.Network/virtualNetworks/VnetName"
                      }
                    ]
```

이 경우 리소스 그룹 및 DDoS 보호 계획의 이름에 해당하는 **ResourceGroupName** 및 **Name** 특성을 모두 지정해야 합니다.

### 예제 2: 리소스 그룹의 모든 DDoS 보호 계획 얻기
