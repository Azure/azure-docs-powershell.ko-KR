# Get-AzApplicationGatewaySslPredefinedPolicy

## SYNOPSIS
Application Gateway에서 제공하는 미리 정의된 SSL 정책을 얻습니다.

## 구문

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Get-AzApplicationGatewaySslPredefinedPolicy** cmdlet은 Application Gateway에서 제공하는 미리 정의된 SSL 정책을 얻습니다.
