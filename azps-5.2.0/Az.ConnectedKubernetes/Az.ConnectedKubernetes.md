# Az.ConnectedKubernetes 모듈
## 설명
Microsoft Azure PowerShell: ConnectedKubernetes cmdlet

## Az.ConnectedKubernetes Cmdlet
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
이름, ID, 속성 및 추가 클러스터 세부 정보를 포함하여 지정된 연결된 클러스터의 속성을 반환합니다.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
새 K8s 클러스터를 등록하여 클러스터에서 추적된 리소스를 만드는 API ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
연결된 클러스터를 삭제하고 Azure Resource Manager에서 추적된 리소스를 제거합니다(ARM.

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
연결된 클러스터 리소스의 특정 속성을 업데이트하는 API
