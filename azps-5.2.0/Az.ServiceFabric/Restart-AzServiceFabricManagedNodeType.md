# Restart-AzServiceFabricManagedNodeType

## SYNOPSIS
노드 형식에서 특정 노드를 다시 시작합니다.

## 구문

```
Restart-AzServiceFabricManagedNodeType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 -NodeName <String[]> [-ForceRestart] [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명
노드 형식에서 특정 노드를 다시 시작합니다. VM을 다시 시작하기 전에 서비스 패브릭 노드를 사용하지 않도록 설정하고 다시 돌아오면 다시 사용하도록 설정합니다. 주 노드 유형에서 이 작업을 수행하면 모든 노드를 동시에 다시 시작하지 않을 수 있습니다. 서비스 패브릭이 노드를 비활성화할 수 없지만 상태 저장 워크로드가 노드에서 실행되는 경우 데이터 손실이 발생할 수 있는 경우에도 -ForceRestart를 사용하여 작업을 강제로 실행합니다.

## 예제

### 예제 1
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Restart-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName  -Name $NodeTypeName -NodeName nt1_0, nt1_3
```

노드 형식에서 노드 0 및 3을 다시 시작합니다.
