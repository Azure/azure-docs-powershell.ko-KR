# Stop-AzStreamAnalyticsJob

## SYNOPSIS
Stream Analytics 작업을 중지합니다.

## 구문

```
Stop-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Stop-AzStreamAnalyticsJob** cmdlet은 Stream Analytics 작업이 Azure에서 실행되지 못하게 비동기적으로 중지하고 사용 중이던 리소스를 삭제합니다.
작업 정의 및 메타데이터는 작업을 편집하고 다시 시작할 수 있도록 Azure Portal 및 관리 API를 통해 구독 내에서 계속 사용할 수 있습니다.
중지됨 상태의 작업 요금은 청구되지 않습니다.

## 예제

### 예제 1: 실행 중인 작업 중지
```powershell
PS C:\>Stop-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

이 명령은 StreamingJob 작업을 중지합니다.
