# Remove-AzConsumptionBudget

## SYNOPSIS
구독 또는 리소스 그룹에서 예산을 제거합니다.

## 구문

### 구독(기본값)
```
Remove-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -Name <String>
 [-ResourceGroupName <String>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### 파이핑
```
Remove-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -InputObject <PSBudget> [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
이 Remove-AzConsumptionBudget cmdlet은 구독 또는 리소스 그룹에서 예산을 제거합니다.

## 예제

### 예제 1: 구독 수준에서 예산 이름을 사용하여 예산 제거
```powershell
PS C:\> Remove-AzConsumptionBudget -Name PSBudget -PassThru
True
```

### 예제 2: 리소스 그룹 수준에서 예산 이름을 사용하여 예산 제거
```powershell
PS C:\> Remove-AzConsumptionBudget -ResourceGroupName RGBudgets -Name PSBudgetRG -PassThru
True
```

### 예제 3: 구독 수준에서 파이핑을 통해 예산 제거
```powershell
PS C:\> Get-AzConsumptionBudget -Name PSBudget | Remove-AzConsumptionBudget -PassThru
True
```
