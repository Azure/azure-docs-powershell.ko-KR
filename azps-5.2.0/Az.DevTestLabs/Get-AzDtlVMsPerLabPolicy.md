# Get-AzDtlVMsPerLabPolicy

## SYNOPSIS
DevTest Labs에서 랩의 랩 정책당 가상 머신을 얻습니다.

## 구문

```
Get-AzDtlVMsPerLabPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzDtlVMsPerLabPolicy** cmdlet은 랩의 랩 정책당 가상 머신을 얻게 하여 랩에 허용되는 총 가상 머신 수를 설정할 수 있습니다.
cmdlet은 정책의 활성화 또는 비활성화 상태 및 정책에서 설정한 랩에서 허용되는 가상 머신의 총 수를 반환합니다.
