# Get-AzDataBoxJob

## SYNOPSIS
Databox 작업 정보를 얻습니다.

## 구문

### ListParameterSet(기본값)
```
Get-AzDataBoxJob [-ResourceGroupName <String>] [-Completed] [-CompletedWithError] [-Cancelled] [-Aborted]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzDataBoxJob -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzDataBoxJob -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzDataBoxJobs** cmdlet은 Azure 구독의 데이터 상자 작업 정보를 얻습니다.
리소스 그룹을 지정하면 이 cmdlet은 해당 리소스 그룹 아래에 있는 모든 데이터 상자 작업을 얻습니다. 리소스 그룹 이름과 함께 작업 이름을 지정하면 이 cmdlet은 해당 특정 데이터 상자 작업 관련 정보를 얻습니다.
구독 ID를 지정하지 않으면 이 cmdlet은 해당 구독의 모든 데이터 상자 작업 관련 정보를 얻습니다.

## 예제

### 예제 1
```powershell
PS C:\> Get-AzDataBoxJob

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
cleanbox                DataBox              Aborted             04-12-2018 16:07:41   westus               TestRg2
cleanbox-Clone          DataBox              Cancelled           25-04-2019 11:31:36   westus               TestRg2
.
.
.
```

Get-AzDataBoxJob 없는 경우 구독에서 모든 데이터 상자 작업을 페치합니다.

### 예제 2
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
.
.
.
```

Get-AzDataBoxJob ResourceGroupName 매개 변수를 사용하여 지정된 리소스 그룹에서 모든 데이터 상자 작업을 페치합니다.

### 예제 3
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1 -Name testtip2

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob ResourceGroupName 및 Name을 사용하여 해당 특정 데이터 상자 작업을 페치합니다.

### 예제 4
```powershell
PS C:\> Get-AzDataBoxJob -ResourceId "/subscriptions/05b5dd1c-793d-41de-be9f-6f9ed142f695/resourceGroups/TestRg1/providers/Microsoft.DataBox/jobs/testtip2"

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob 지정한 경우 해당 특정 데이터 상자 작업을 페치합니다.
