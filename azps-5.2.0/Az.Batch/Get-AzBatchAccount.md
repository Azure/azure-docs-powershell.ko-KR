# Get-AzBatchAccount

## SYNOPSIS
현재 구독에서 Batch 계정을 얻습니다.

## 구문

```
Get-AzBatchAccount [[-AccountName] <String>] [[-ResourceGroupName] <String>] [[-Tag] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzBatchAccount** cmdlet은 현재 구독에서 Azure Batch 계정을 얻습니다. *AccountName* 매개 변수를 사용하여 단일 계정을 얻거나 *ResourceGroupName* 매개 변수를 사용하여 해당 리소스 그룹에서 계정을 얻을 수 있습니다.

## 예제

### 예제 1: 이름에 따라 일괄 처리 계정 얻기
```
PS C:\>Get-AzBatchAccount -AccountName "pfuller"
AccountName                  : pfuller
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://pfuller.westus.batch.azure.com
```

이 명령은 pfuller라는 일괄 처리 계정을 얻습니다.

### 예제 2: 리소스 그룹과 연결된 배치 계정 얻기
```
PS C:\>Get-AzBatchAccount -ResourceGroupName "CmdletExampleRG"
AccountName                  : cmdletexample
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
AccountName                  : cmdletexample2
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

이 명령은 CmdletExampleRG 리소스 그룹과 연결된 배치 계정을 얻습니다.
