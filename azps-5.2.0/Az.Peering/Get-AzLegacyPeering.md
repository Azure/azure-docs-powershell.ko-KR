# Get-AzLegacyPeering

## SYNOPSIS
레거시 피어링 리소스를 Azure 리소스 관리(ARM) 리소스로 변환하는 데 사용됩니다. 

## 구문

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
이 명령은 Azure 리소스 관리(ARM) 리소스로 변환할 레거시 피어링 리소스를 보는 데 사용됩니다.
