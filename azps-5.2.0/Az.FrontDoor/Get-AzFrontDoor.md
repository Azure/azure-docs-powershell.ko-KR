# Get-AzFrontDoor

## SYNOPSIS
Front Door 부하 균형 조정기

## 구문

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Get-AzFrontDoor** cmdletGet은 제공된 정보와 일치하는 현재 구독의 모든 기존 Front Door를 얻습니다.
