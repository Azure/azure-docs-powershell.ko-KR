# Set-AzCdnProfile

## SYNOPSIS
CDN 프로필을 업데이트합니다.

## 구문

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Set-AzCdnProfile** cmdlet은 Azure CDN(Content Delivery Network) 프로필을 업데이트합니다.
