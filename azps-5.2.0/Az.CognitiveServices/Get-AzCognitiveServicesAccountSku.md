# Get-AzCognitiveServicesAccountSku

## SYNOPSIS
계정에 사용 가능한 SKUS를 얻습니다.

## 구문

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzCognitiveServicesAccountSku** cmdlet은 Cognitive Services 계정에 사용 가능한 SKU를 얻습니다.
SKU는 계정에 대한 계층 계획입니다.
계정에 대한 가격, 호출 제한 및 요금을 정의합니다.
F0 SKU는 무료 계층입니다.
유료 계층에는 S0, S1, S2 등이 포함됩니다.

## 예제

### 예제 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```
