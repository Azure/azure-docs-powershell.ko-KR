# Select-AzContext

## SYNOPSIS
Azure PowerShell cmdlet에서 대상으로 할 구독 및 계정 선택

## 구문

### SelectByInputObject(기본값)
```
Select-AzContext -InputObject <PSAzureContext> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SelectByName
```
Select-AzContext [-Scope <ContextModificationScope>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [-Name] <String> [<CommonParameters>]
```

## 설명
Azure PowerShell cmdlet에서 대상(또는 계정 또는 테넌트)에 대한 구독을 선택합니다.  이 cmdlet 후 이후 cmdlet은 선택한 컨텍스트를 대상으로 합니다.

## 예제

### 예제 1: 명명된 컨텍스트 대상 지정
```powershell
PS C:\> Select-AzContext "Work"

Name    Account             SubscriptionName    Environment         TenantId
----    -------             ----------------    -----------         --------
Work    test@outlook.com    Subscription1       AzureCloud          xxxxxxxx-x...
```

'Work' 컨텍스트의 계정, 테넌트 및 구독에서 향후 Azure PowerShell cmdlet을 대상으로 합니다.

### 예제 2

이 PowerShell 세션에서 Azure PowerShell cmdlet을 대상으로 할 구독 및 계정을 선택합니다. (자동Generated)

```powershell <!-- Aladdin Generated Example --> 
Select-AzContext -Name TestEnvironment -Scope Process
```
