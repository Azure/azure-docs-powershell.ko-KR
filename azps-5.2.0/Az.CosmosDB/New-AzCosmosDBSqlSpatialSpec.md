# New-AzCosmosDBSqlSpatialSpec

## SYNOPSIS
PSSpatialSpec 형식의 새 개체를 만듭니다. Set-AzCosmosDBSqlIndexingPolicy에 대한 매개 변수 값으로 전달할 수 있습니다.

## 구문

```
New-AzCosmosDBSqlSpatialSpec -Path <String> -Type <String[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
Sql API의 SpatialSpec에 해당하는 개체를 만듭니다.

## 예제

### 예제 1
```powershell
PS C:\> New-AzCosmosDBSqlSpatialSpec -Path "/abc" -Type String
Path Types
---- -----
/abc {String}
```
