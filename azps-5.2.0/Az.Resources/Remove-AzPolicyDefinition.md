# Remove-AzPolicyDefinition

## SYNOPSIS
정책 정의를 제거합니다.

## 구문

```

```

```
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzPolicyDefinition** cmdlet은 정책 정의를 제거합니다.

## 예제

### 예제 1: 이름으로 정책 정의 제거
```
PS C:\> Remove-AzPolicyDefinition -Name 'VMPolicyDefinition'
```

이 명령은 지정된 정책 정의를 제거합니다.

### 예제 2: 리소스 ID로 정책 정의 제거
```
PS C:\> $PolicyDefinition = Get-AzPolicyDefinition -Name 'VMPolicyDefinition' 
PS C:\> Remove-AzPolicyDefinition -Id $PolicyDefinition.ResourceId -Force
```

첫 번째 명령은 Get-AzPolicyDefinition cmdlet을 사용하여 VMPolicyDefinition이라는 정책 정의를 얻습니다.
이 명령은 $PolicyDefinition 저장합니다.
두 번째 명령은 리소스의 **ResourceId** 속성으로 식별되는 정책 정의를 $PolicyDefinition.
