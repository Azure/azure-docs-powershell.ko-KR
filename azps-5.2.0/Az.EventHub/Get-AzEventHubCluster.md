# Get-AzEventHubCluster

## SYNOPSIS
단일 이벤트 허브 클러스터의 세부 정보를 얻거나 이벤트 허브 클러스터 목록을 얻습니다.

## 구문

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
이 Get-AzEventHubCluster cmdlet은 이벤트 허브 클러스터의 세부 정보 또는 주어진 리소스 그룹의 모든 Event Hub 클러스터 목록을 반환합니다.
클러스터 이름이 제공된 경우 단일 클러스터의 세부 정보가 반환됩니다.
클러스터 이름이 제공되지 않으면 지정된 리소스 그룹의 모든 클러스터 목록이 반환됩니다.

```powershell

```
