# Get-AzIotHubEventHubConsumerGroup

## SYNOPSIS
모든 eventhub 소비자group을 얻습니다.

## 구문

```
Get-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
IotHub에서 사용하는 다른 EventHubs에 대한 모든 eventhub 소비자group을 얻습니다.

## 예제

### 예제 1은 원격 분석 eventhub에 대한 모든 eventhub 소비자group을 얻습니다.
```
PS C:\> Get-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

myiothub라는 iothub에 대한 원격 분석 eventhub에 대한 모든 eventhub 소비자group을 얻습니다.
