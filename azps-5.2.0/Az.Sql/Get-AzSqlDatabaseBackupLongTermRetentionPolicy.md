# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
데이터베이스 장기 보존 정책을 얻습니다.

## 구문

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Get-AzSqlDatabaseBackupLongTermRetentionPolicy** cmdlet은 이 데이터베이스에 등록된 장기 보존 정책을 얻습니다.
정책은 백업 저장소 정책을 정의하는 데 사용되는 Azure Backup 리소스입니다.

## 예제

### 예제 1: 현재 버전의 장기 보존 정책 다운로드
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

이 명령은 database01에 대한 장기 보존 정책의 현재 버전을 얻습니다.
