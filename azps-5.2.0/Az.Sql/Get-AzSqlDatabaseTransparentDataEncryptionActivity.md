# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
데이터베이스의 TDE 검사 진행률을 얻습니다.

## 구문

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Get-AzSqlDatabaseTransparentDataEncryptionActivity** cmdlet은 Azure SQL 데이터베이스의 TDE(투명한 데이터 암호화) 검사의 진행률을 얻습니다.
암호화 범위가 실행되고 있는 것이 없는 경우 이 cmdlet은 빈 목록을 반환합니다.
자세한 내용은 Azure SQL Database(Microsoft Developer Network 라이브러리)를 사용하여 https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) 투명한 데이터 암호화를 참조하세요.
이 cmdlet은 Azure의 SQL Server Stretch Database 서비스에서도 지원됩니다.

## 예제

### 예제 1: 데이터베이스에 대한 TDE 작업 얻기
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

이 명령은 server01이라는 서버에서 database01이라는 데이터베이스에 대한 TDE 활동을 얻습니다.
