# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
권한 부여 서버를 제거합니다.

## 구문

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzApiManagementAuthorizationServer** cmdlet은 Azure API Management 권한 부여 서버를 제거합니다.

## 예제

### 예제 1: 권한 부여 서버 제거
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

이 명령은 지정된 API Management 권한 부여 서버를 제거합니다.
Force 매개 *변수가* 지정되어 있기 때문에 확인이 필요하지 않습니다.
