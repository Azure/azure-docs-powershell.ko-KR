# Remove-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
애플리케이션 게이트웨이에서 자동 조정 구성을 제거합니다.

## 구문

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzApplicationGatewayAutoscaleConfiguration** cmdlet은 기존 Application Gateway에서 자동 크기 조정 구성을 제거합니다.

## 예제

### 예제 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

첫 번째 명령은 애플리케이션 게이트웨이를 사용하여 애플리케이션 게이트웨이를 $gw 저장합니다.
두 번째 명령은 애플리케이션 게이트웨이에서 자동 조정 구성을 제거합니다.
세 번째 명령은 Azure에서 애플리케이션 게이트웨이를 업데이트합니다.
