# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
애플리케이션 게이트웨이에 대한 프런트 엔드 포트를 만듭니다.

## 구문

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**New-AzApplicationGatewayFrontendPort** cmdlet은 Azure 애플리케이션 게이트웨이에 대한 프런트 엔드 포트를 만듭니다.

## 예제

### 예제 1: Example1: 프런트 엔드 포트 만들기
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

이 명령은 포트 80에 FrontEndPort01이라는 프런트 엔드 포트를 만들고, 그 결과를 $FrontEndPort.
