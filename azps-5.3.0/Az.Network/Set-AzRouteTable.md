# Set-AzRouteTable

## SYNOPSIS
경로 테이블을 업데이트합니다.

## 구문

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명
**Set-AzRouteTable** cmdlet은 경로 테이블을 업데이트합니다.

### 예제 2: 경로 테이블 수정

```
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
False
PS C:\> $rt.DisableBgpRoutePropagation = $true
PS C:\> Set-AzRouteTable -RouteTable $rt
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
True
```

첫 번째 명령은 rtName이라는 경로 테이블을 사용하여 $rt 변수에 저장합니다.
두 번째 명령은 DisableBgpRoutePropagation의 값을 표시합니다.
세 번째 명령은 DisableBgpRoutePropagation의 값을 업데이트합니다.
네 번째 명령은 서버의 경로 테이블을 업데이트합니다.
다섯 번째 명령은 업데이트된 경로 테이블을 다운로드하여 $rt 저장합니다.
여섯 번째 명령은 DisableBgpRoutePropagation의 값을 표시합니다.
