# Update-AzSentinelBookmark

## SYNOPSIS
책갈피를 업데이트합니다.

## 설명
**Update-AzSentinelBookmark** cmdlet은 지정된 작업 영역의 책갈피를 업데이트합니다.
**Bookmark** 개체를 매개 변수로 전달하거나 파이프라인 연산자를 사용하여 전달하거나 필요한 *BookmarkId* 매개 변수를 지정할 수 있습니다.
*Confirm* 매개 변수 및 $ConfirmPreference Windows PowerShell cmdlet에서 확인을 요청하는지 여부를 제어할 수 있습니다.

## 예제

### 예제 1
```powershell
PS C:\> Update-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceNAme" -BookmarkId "MyBookmarkId" -Notes "Found something interesting"
```

이 명령은 Notes 속성을 설정하여 책갈피를 *업데이트합니다.*  다른 모든 propreties는 동일하게 유지.

### 예제 2
```powershell
PS C:\> $Bookmark = Get-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceNAme" -BookmarkId "MyBookmarkId"
PS C:\> $Bookmark | Set-AzSentinelBookmark -Notes "Found something interesting"
```

첫 번째 명령은 지정된 작업 영역의 *BookmarkId로* 책갈피를 읽은 다음 $Bookmark 저장합니다.
두 번째 명령은 Notes 속성을 업데이트합니다.   다른 모든 propreties는 동일하게 유지.
