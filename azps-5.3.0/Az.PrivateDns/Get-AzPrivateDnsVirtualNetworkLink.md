# Get-AzPrivateDnsVirtualNetworkLink

## SYNOPSIS
지정된 개인 DNS 영역과 연결된 가상 네트워크 링크를 얻습니다.

## 구문

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzPrivateDnsVirtualNetworkLink** cmdlet은 지정된 리소스 그룹에서 특정 개인 DNS 영역과 연결된 가상 네트워크 링크를 얻습니다.
Name 매개 *변수를 지정하면* 단일 **PSPrivateDnsVirtualNetworkLink 개체가** 반환됩니다.
이름 매개 변수를  지정하지 않으면 지정된 리소스 그룹의 영역과 연결된 모든 링크를 포함하는 배열이 반환됩니다.
**PSPrivateDnsVirtualNetworkLink** 개체를 사용하여 링크를 업데이트할 수 있습니다.

The link object returned looks like the following:

ZoneName                : myzone.com

                          etwork/virtualNetworks/myvirtualnetwork
Location                :

Tags                    : {tag1}
RegistrationEnabled     : True
