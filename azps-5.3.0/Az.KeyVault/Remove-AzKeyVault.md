# Remove-AzKeyVault

## SYNOPSIS
키 자격 증명 모음을 삭제합니다.

## 구문

### ByAvailableVault(기본값)
```
Remove-AzKeyVault [-VaultName] <String> [[-ResourceGroupName] <String>] [[-Location] <String>] [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByDeletedVault
```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

```

```

```

```


## 설명
**Remove-AzKeyVault** cmdlet은 지정된 키 자격 증명 모음을 삭제합니다.
또한 해당 인스턴스에 포함된 모든 키와 비밀도 삭제합니다.
이 cmdlet에 대해 리소스 그룹을 지정하는 것은 선택 사항이지만 성능을 향상하기 위해 이렇게 해야 합니다.

## 예제

### 예제 1: 키 자격 증명 모음 제거
```powershell
PS C:\> Remove-AzKeyVault -VaultName "Contoso03Vault" -PassThru

True
```

이 명령은 현재 구독에서 Contoso03Vault라는 키 자격 증명 모음을 제거합니다.

### 예제 2: 지정된 리소스 그룹에서 키 자격 증명 모음 제거
```powershell
PS C:\> Remove-AzKeyVault -Name "Contoso03Vault" -ResourceGroupName "Group14" -PassThru

True
```

이 명령은 명명된 리소스 그룹에서 Contoso03Vault라는 키 자격 증명 모음을 제거합니다.
리소스 그룹 이름을 지정하지 않으면 cmdlet은 현재 구독에서 삭제할 명명된 키 자격 증명 모음을 검색합니다.

### 예제 3: 관리되는 hsm 제거
```powershell
PS C:\>  Remove-AzKeyVault -Name "testManagedHsm" -Hsm -PassThru

True
```

이 명령은 현재 구독에서 testManagedHsm이라는 관리되는 hsm을 제거합니다.
