# Get-AzApiManagementSsoToken

## SYNOPSIS
API Management 서비스의 배포된 관리 포털에 SSO 토큰이 있는 링크를 얻습니다.

## 구문

### ExpandedParameter(기본값)
```
Get-AzApiManagementSsoToken -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObject
```
Get-AzApiManagementSsoToken -InputObject <PsApiManagement> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Get-AzApiManagementSsoToken** cmdlet은 SSO(Single Sign-On) 토큰을 포함하는 링크(URL)를 API Management 서비스의 배포된 관리 포털에 반환합니다.

## 예제

### 예제 1: API Management 서비스의 SSO 토큰을 얻습니다.
```
PS C:\>Get-AzApiManagementSsoToken -ResourceGroupName "Contoso" -Name "ContosoApi"
```

이 명령은 API Management 서비스의 SSO 토큰을 얻습니다.
