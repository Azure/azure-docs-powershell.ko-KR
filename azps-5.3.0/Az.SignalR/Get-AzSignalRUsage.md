# Get-AzSignalRUsage

## SYNOPSIS
구독의 사용 할당량을 얻습니다.

## 구문

```
Get-AzSignalRUsage [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
구독의 사용 할당량을 얻습니다.

## 예제

### 위치를 입력하여 사용 할당량 얻음
```powershell
PS C:\> Get-AzSignalRUsage eastus

Name                 CurrentValue Limit
----                 ------------ -----
FreeTierInstances    2            5
SignalRTotalUnits    3            300
```
