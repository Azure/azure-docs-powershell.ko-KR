# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
계정에 대한 API 키를 얻습니다.

## 구문

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzCognitiveServicesAccountKey** cmdlet은 프로비전된 Cognitive Services 계정에 대한 API 키를 얻습니다.
Cognitive Services 계정에는 Key1 및 Key2의 두 개의 API 키가 있습니다.
키를 사용하면 Cognitive Services 계정 엔드포인트와 상호 작용할 수 있습니다.
키를 New-AzCognitiveServicesAccountKey 키를 다시 생성하는 데 사용할 수 있습니다.

## 예제

### 예제 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
