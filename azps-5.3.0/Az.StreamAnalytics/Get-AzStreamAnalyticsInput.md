# Get-AzStreamAnalyticsInput

## SYNOPSIS
Stream Analytics 작업 입력을 얻습니다.

## 구문

```
Get-AzStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzStreamAnalyticsInput** cmdlet은 Stream Analytics 작업에서 정의된 모든 입력을 나열하거나 특정 입력에 대한 정보를 얻습니다.

## 예제

### 예제 1: 작업에서 정의된 입력에 대한 정보 얻기
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

이 명령은 StreamingJob 작업에서 정의된 모든 입력에 대한 정보를 반환합니다.

### 예제 2: 작업에서 정의된 특정 입력에 대한 정보 얻기
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

이 명령은 StreamingJob 작업에서 정의된 EntryStream이라는 입력에 대한 정보를 반환합니다.
