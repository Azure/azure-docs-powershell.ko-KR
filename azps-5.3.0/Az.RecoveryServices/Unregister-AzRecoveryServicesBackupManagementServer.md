# Unregister-AzRecoveryServicesBackupManagementServer

## SYNOPSIS
자격 증명 모음에서 SCDPM 서버 또는 Backup 서버를 등록을 등록하지 않습니다.

## 구문

```
Unregister-AzRecoveryServicesBackupManagementServer [-AzureRmBackupManagementServer] <BackupEngineBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Unregister-AzRecoveryServicesBackupManagementServer** cmdlet은 자격 증명 모음에서 SCDPM(System Center Data Protection Manager) 서버 또는 Azure Backup 서버의 등록을 등록을 해지합니다.
이 cmdlet은 자격 증명 모음에서 등록되지 않은 서버에 대한 참조를 제거합니다.
컨테이너를 등록을 등록을 등록하려면 먼저 해당 컨테이너와 연결된 모든 보호된 데이터를 삭제해야 합니다.
현재 cmdlet을 사용하기 전에 Set-AzRecoveryServicesVaultContext cmdlet을 사용하여 자격 증명 모음 컨텍스트를 설정할 수 있습니다.

## 예제

### 예제 1: 자격 증명 모음에서 SCDPM 서버 등록을 등록하지 않습니다.
```powershell
PS C:\>$BMS = Get-AzRecoveryServicesBackupManagementServer -Name "dpmserver01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupManagementServer -AzBackupManagementServer $BMS
```

첫 번째 명령은 dpmserver01.contoso.com Backup 관리 서버를 $BMS 변수에 저장합니다.
두 번째 명령은 자격 증명 모음에서 SCDPM 서버를 등록을 등록합니다.

### 예제 2

자격 증명 모음에서 SCDPM 서버 또는 Backup 서버를 등록을 등록하지 않습니다. (자동Generated)
