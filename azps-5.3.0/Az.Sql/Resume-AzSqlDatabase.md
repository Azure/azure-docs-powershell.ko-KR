# Resume-AzSqlDatabase

## SYNOPSIS
데이터 웨어하우스 SQL 다시 시작됩니다.

## 구문

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Resume-AzSqlDatabase** cmdlet은 Azure SQL Data Warehouse 데이터베이스를 다시 시작됩니다.

## 예제

### 예제 1: Azure SQL Data Warehouse 데이터베이스 다시 시작
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

이 명령은 일시 중단된 Azure SQL Data Warehouse 데이터베이스를 다시 시작합니다.
