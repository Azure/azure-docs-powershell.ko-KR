# Enable-AzBatchTask

## SYNOPSIS
작업을 다시 활성화합니다.

## 구문

### ID
```
Enable-AzBatchTask [-JobId] <String> [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Enable-AzBatchTask [-Task] <PSCloudTask> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Enable-AzBatchTask** cmdlet은 작업을 다시 활성화합니다.
태스크가 재시도 횟수를 소진한 경우 이 cmdlet을 사용하면 실행할 수 있습니다.

## 예제

### 예제 1: 작업 다시 활성화
```
PS C:\>Enable-AzBatchTask -JobId "Job7" -Id "Task2" -BatchContext $Context
```

이 명령은 Job7에서 task Task2를 다시 활성화합니다.

### 예제 2: 파이프라인을 사용하여 작업 다시 활성화
```
PS C:\>Get-AzBatchTask -JobId "Job8" -Id "Task3" -BatchContext $Context | Enable-AzBatchTask -BatchContext $Context
```

이 명령은 Get-AzBatchTask cmdlet을 사용하여 ID Job8이 있는 작업에 ID Task3이 있는 Batch 태스크를 Get-AzBatchTask 합니다.
이 명령은 파이프라인 연산자를 사용하여 이 작업을 현재 cmdlet에 전달합니다.
이 명령은 이 작업을 다시 활성화합니다.
