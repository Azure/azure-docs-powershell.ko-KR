# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Data Lake Store 계정을 영구적으로 삭제합니다.

## 구문

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzDataLakeStoreAccount** cmdlet은 Data Lake Store 계정을 영구적으로 삭제합니다.

## 예제

### 예제 1: Data Lake Store 계정 제거
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

이 명령은 Data Lake Store에서 ContosoADL이라는 계정을 제거합니다.
