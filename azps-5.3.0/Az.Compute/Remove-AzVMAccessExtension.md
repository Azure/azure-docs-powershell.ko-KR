# Remove-AzVMAccessExtension

## SYNOPSIS
가상 머신에서 VMAccess 확장을 제거합니다.

## 구문

```
Remove-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzVMAccessExtension** cmdlet은 가상 머신에서 VMAccess(Virtual Machine Access) Virtual Machine 확장을 제거합니다.

## 예제

### 예제 1

가상 머신에서 VMAccess 확장을 제거합니다. (자동Generated)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzVMAccessExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
