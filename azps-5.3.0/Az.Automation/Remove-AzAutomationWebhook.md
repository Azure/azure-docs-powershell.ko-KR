# Remove-AzAutomationWebhook

## SYNOPSIS
Automation Runbook에서 웹후크를 제거합니다.

## 구문

```
Remove-AzAutomationWebhook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzAutomationWebhook** cmdlet은 Azure Automation Runbook에서 웹후크를 제거합니다.
웹후크가 삭제됩니다.

## 예제

### 예제 1: 웹후크 제거
```
PS C:\>Remove-AzAutomationWebhook -Name "Webhook11" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Force
```

이 명령은 AutomationAccount01이라는 Automation 계정에서 Webhook11이라는 웹후크를 제거합니다.
이 명령은 Force 매개 *변수를 지정합니다.*
따라서 확인을 묻는 메시지가 표시되지 않습니다.
