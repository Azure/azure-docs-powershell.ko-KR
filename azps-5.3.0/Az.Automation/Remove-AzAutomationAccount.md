# Remove-AzAutomationAccount

## SYNOPSIS
Automation 계정을 제거합니다.

## 구문

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzAutomationAccount** cmdlet은 리소스 그룹에서 Azure Automation 계정을 제거합니다.
Automation 계정에 대한 자세한 내용은 다음 cmdlet을 New-AzAutomationAccount 참조하세요.

## 예제

### 예제 1: 자동화 계정 제거
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

이 명령은 사용자 유효성 검사를 요청하지 않고 ContosoAutomationAccount라는 자동화 계정을 제거합니다.
