# Suspend-AzAutomationJob

## SYNOPSIS
Automation 작업을 일시 중단합니다.

## 구문

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Suspend-AzAutomationJob** cmdlet은 Azure Automation 작업을 일시 중단합니다.
실행 중인 Automation 작업을 지정합니다.
일시 중단된 작업을 다시 시작하기 위해 Resume-AzAutomationJob cmdlet을 사용 합니다.

## 예제

### 예제 1: 작업 일시 중단
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

이 명령은 지정된 ID가 있는 작업을 일시 중단합니다.
