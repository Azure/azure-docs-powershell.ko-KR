# Disable-AzHDInsightMonitoring

## SYNOPSIS
HDInsight 클러스터에서 모니터링을 사용하지 않도록 설정하면 관련 로그가 활성화 중에 지정된 모니터링 작업 영역으로 흐르는 것을 중지합니다.

## 구문

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Disable-AzHDInsightMonitoring** cmdlet은 Azure HDInsight 클러스터에서 모니터링을 사용하지 않도록 설정합니다.

## 예제

### 예제 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

HDInsight 클러스터에서 모니터링을 사용하지 않도록 설정하고 관련 로그가 모니터링 작업 영역으로 흐르지 않습니다.

### 예제 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

HDInsight 클러스터에서 모니터링을 사용하지 않도록 설정하고 관련 로그가 모니터링 작업 영역으로 흐르지 않습니다.
