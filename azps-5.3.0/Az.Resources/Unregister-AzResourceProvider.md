# Unregister-AzResourceProvider

## SYNOPSIS
리소스 공급자를 등록을 등록하지 않습니다.

## 구문

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Unregister-AzResourceProvider** cmdlet은 Azure 리소스 공급자를 등록을 등록하지 않습니다.

## 예제

### 예제 1: ProviderNamespace를 통해 리소스 공급자 등록을 등록하지 않습니다.

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

이 명령은 리소스 공급자 "Microsoft.support"의 등록을 등록을 등록하지 않습니다.
