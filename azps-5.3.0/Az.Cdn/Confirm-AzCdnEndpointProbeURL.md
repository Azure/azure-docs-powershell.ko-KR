# Confirm-AzCdnEndpointProbeURL

## SYNOPSIS
프로브 URL의 유효성을 검사합니다.

## 구문

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Confirm-AzCdnEndpointProbeURL** cmdlet은 제공된 프로브 URL을 동적 사이트 가속에 사용할 수 있는지 확인합니다.

## 예제

### 예제 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

프로브 URL " "의 http://www.bing.com/images 유효성을 검사합니다.
