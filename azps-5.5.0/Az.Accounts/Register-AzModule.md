# Register-AzModule

## SYNOPSIS
내부 사용 전용 - 생성된 자동Rest cmdlet에 대한 런타임 지원 제공

## 구문

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
내부 사용 전용 - 생성된 자동Rest cmdlet에 대한 런타임 지원 제공

## 예제

### 예제 1
```powershell
PS C:\> Register-AzModule
```

AutoRest에서 생성된 cmdlet에서 내부적으로 사용
