# Get-AzVmssSku

## SYNOPSIS
VMSS에 사용 가능한 SKUS를 얻습니다.

## 구문

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzVmssSku** cmdlet은 VMSS(Virtual Machine Scale Set)에 사용 가능한 SKU를 얻습니다.

## 예제

### 예제 1: VMSS에서 사용 가능한 모든 SKUS를 얻습니다.
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

이 명령은 ContosoGroup이라는 리소스 그룹에 속하는 ContosoVMSS라는 VMSS에서 사용 가능한 모든 SKUS를 얻습니다.
