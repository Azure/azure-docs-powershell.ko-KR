# Get-AzAvailabilitySet

## SYNOPSIS
리소스 그룹의 Azure 가용성 집합을 얻습니다.

## 구문

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzAvailabilitySet** cmdlet은 리소스 그룹에서 Azure 가용성 집합을 얻습니다.
얻을 특정 가용성 집합의 이름을 지정할 수 있습니다.

## 예제

### 예제 1: 특정 가용성 집합을 구합니다.
