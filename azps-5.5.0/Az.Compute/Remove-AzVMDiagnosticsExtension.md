# Remove-AzVMDiagnosticsExtension

## SYNOPSIS
가상 머신에서 진단 확장을 제거합니다.

## 구문

```
Remove-AzVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Remove-AzVMDiagnosticsExtension** cmdlet은 가상 머신에서 Azure Diagnostics 확장을 제거합니다.
변경 내용을 구현하려면 이 cmdlet의 출력을 Update-AzVM cmdlet에 전달해야 합니다.

## 예제

### 예제 1: 가상 머신에서 진단 확장 제거
```
PS C:\> Remove-AzVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" | Update-AzVM
```

이 명령은 ContosoVM22라는 가상 머신에서 진단 확장을 제거합니다.
이 명령은 파이프라인 연산자를 사용하여 Update-AzVM cmdlet에 결과를 전달합니다.
이 명령은 가상 머신을 업데이트합니다.
