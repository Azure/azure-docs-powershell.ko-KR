# Get-AzSnapshot

## SYNOPSIS
스냅숏의 속성을 얻습니다.

## 구문

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzSnapshot** cmdlet은 스냅숏의 속성을 얻습니다.
