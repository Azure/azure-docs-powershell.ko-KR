# Start-AzVmss

## SYNOPSIS
VMSS 또는 VMSS 내의 가상 머신 집합을 시작합니다.

## 구문

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Start-AzVmss** cmdlet은 VMSS(Virtual Machine Scale Set) 또는 가상 머신 집합 내의 모든 가상 머신을 시작합니다.
InstanceId 매개 *변수를* 사용하여 가상 머신 집합을 선택할 수 있습니다.

## 예제

### 예제 1: VMSS 내에서 특정 가상 머신 집합 시작
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

이 명령은 ContosoVMSS라는 VMSS에 속하는 인스턴스 ID 문자열 배열로 지정된 특정 가상 머신 집합을 시작합니다.

### 예제 2: VMSS 내의 모든 가상 머신 시작
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

이 명령은 ContosoVMSS라는 VMSS에 속하는 모든 가상 머신을 시작합니다.
