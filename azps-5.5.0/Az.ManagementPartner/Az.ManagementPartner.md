# Az.ManagementPartner 모듈
## 설명
Azure Resource Manager용 관리 파트너 cmdlet입니다.

## Az.ManagementPartner Cmdlet
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
현재 인증된 사용자 또는 서비스 주체의 MPN(Microsoft 파트너 네트워크) ID를 얻습니다. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
현재 인증된 사용자 또는 서비스 주체에 MPN(Microsoft 파트너 네트워크) ID를 연결합니다.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
현재 인증된 사용자 또는 서비스 주체의 MPN(Microsoft 파트너 네트워크) ID를 삭제합니다.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
현재 인증된 사용자 또는 서비스 주체의 MPN(Microsoft 파트너 네트워크) ID를 업데이트합니다.
