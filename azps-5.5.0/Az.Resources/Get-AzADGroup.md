# Get-AzADGroup

## SYNOPSIS
Active Directory 그룹을 필터합니다.

## 구문

### EmptyParameterSet(기본값)
```
Get-AzADGroup [-ObjectId <Guid>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### SearchStringParameterSet
```
Get-AzADGroup -DisplayNameStartsWith <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzADGroup -DisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Get-AzADGroup -ObjectId <Guid> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount] [-Skip <UInt64>]
 [-First <UInt64>] [<CommonParameters>]
```

## 설명
Active Directory 그룹을 필터합니다.

## 예제

### 예제 1: 모든 AD 그룹 나열
```powershell
PS C:\> Get-AzADGroup
```

테넌트의 모든 AD 그룹을 나열합니다.

### 예제 2: Paging를 사용하여 모든 AD 그룹 나열

```powershell
PS C:\> Get-AzADGroup -First 100
```

테넌트의 처음 100개 AD 그룹을 나열합니다.

### 예제 3: 개체 ID로 AD 그룹 얻기

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

개체 ID가 '85F89C90-780E-4AA6-9F4F-6F268D322EEE'인 AD 그룹을 얻습니다.

### 예제 4: 검색 문자열로 그룹 나열

```powershell
PS C:\> Get-AzADGroup -SearchString Joe
```

표시 이름이 'Joe'로 시작하는 모든 AD 그룹을 나열합니다.
