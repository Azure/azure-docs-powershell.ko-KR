# New-AzUserAssignedIdentity

## SYNOPSIS
새 사용자 할당 ID를 생성하거나 기존 사용자 할당 ID를 업데이트합니다.

## 구문

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**New-AzUserAssignedIdentity** cmdlet은 새 사용자 할당 ID를 만듭니다. 기존 ID와 함께 사용하는 경우 ID를 업데이트했습니다.
ID에 Azure Resource Manager 태그를 추가하기 위해 Set-AzResource cmdlet을 사용하세요.
