# Get-AzCognitiveServicesAccount

## SYNOPSIS
계정을 얻습니다.

## 구문

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzCognitiveServicesAccount** cmdlet은 *ResourceGroupName* 매개 변수로 지정된 리소스 그룹에서 프로비전된 Cognitive Services 계정을 얻습니다.
*ResourceGroupName* 매개 변수를 지정하지 않으면 이 cmdlet은 현재 구독에 대한 모든 Cognitive Services 계정을 얻습니다.
