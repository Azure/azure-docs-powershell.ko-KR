# Get-AzCloudServiceInstanceView

## SYNOPSIS
클라우드 서비스의 상태를 얻습니다.

## 구문

```
Get-AzCloudServiceInstanceView -CloudServiceName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 설명
클라우드 서비스의 상태를 얻습니다.

## 예제

### 예제 1: 클라우드 서비스 인스턴스 보기를 얻습니다.
```powershell

PS C:\>$cloudServiceInstanceView

    "statusesSummary": [
      {
        "code": "ProvisioningState/succeeded",

      }
    ]
  },
  "statuses": [
    {
      "code": "ProvisioningState/succeeded",
```

이 cmdlet은 ContosOrg라는 리소스 그룹에 속하는 ContosoCS라는 클라우드 서비스의 인스턴스 보기를 얻습니다.
