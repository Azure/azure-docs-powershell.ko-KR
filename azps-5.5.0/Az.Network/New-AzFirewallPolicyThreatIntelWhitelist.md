# New-AzFirewallPolicyThreatIntelWhitelist

## SYNOPSIS
Azure Firewall 정책에 대한 새 위협 인텔리전스 허용 목록 만들기

## 구문

```
New-AzFirewallPolicyThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**New-AzFirewallPolicyThreatIntelWhitelist** cmdlet은 Azure Firewall 정책을 만들거나 설정할 때 사용할 수 있는 위협 Intel 허용 목록 개체를 만듭니다.

## 예제

### 예제 1
```powershell
PS C:\> New-AzFirewallPolicyThreatIntelWhitelist -IpAddress 23.46.72.91,192.79.236.79 -FQDN microsoft.com
```

이 예제에서는 하나의 항목의 FQDN 화이트리스트와 두 개의 항목으로 된 IP 주소 화이트리스트를 포함하는 위협 Intel 화이트리스트를 만듭니다.
