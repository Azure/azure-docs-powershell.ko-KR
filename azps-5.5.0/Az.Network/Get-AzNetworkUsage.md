# Get-AzNetworkUsage

## SYNOPSIS
구독에 대한 네트워크 사용량을 나열합니다.

## 구문

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
이 Get-AzNetworkUsage cmdlet은 네트워크 리소스에 대한 제한 및 현재 사용량을 얻습니다.
