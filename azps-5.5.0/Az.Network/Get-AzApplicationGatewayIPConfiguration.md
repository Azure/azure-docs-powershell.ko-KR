# Get-AzApplicationGatewayIPConfiguration

## SYNOPSIS
애플리케이션 게이트웨이의 IP 구성을 얻습니다.

## 구문

```
Get-AzApplicationGatewayIPConfiguration [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzApplicationGatewayIPConfiguration** cmdlet은 애플리케이션 게이트웨이의 IP 구성을 얻습니다.
IP 구성에는 애플리케이션 게이트웨이가 배포된 서브넷이 포함되어 있습니다.

## 예제

### 예제 1: 특정 IP 구성을 갖기
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $GatewaySubnet = Get-AzApplicationGatewayIPConfiguration -Name "GatewaySubnet01" -ApplicationGateway $AppGw
```

첫 번째 명령은 애플리케이션 게이트웨이를 사용하여 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다. 두 번째 명령은 게이트웨이에 저장된 게이트웨이에서 GateSubnet01이라는 IP 구성을 $AppGw.

### 예제 2: IP 구성 목록 확인
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $GatewaySubnets = Get-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw
```

첫 번째 명령은 애플리케이션 게이트웨이를 사용하여 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다. 두 번째 명령은 모든 IP 구성의 목록을 얻습니다.
