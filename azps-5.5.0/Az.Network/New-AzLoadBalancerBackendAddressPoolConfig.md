# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
부하 균형 조정에 대한 백end 주소 풀 구성을 만듭니다.

## 구문

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명
**New-AzLoadBalancerBackendAddressPoolConfig** cmdlet은 Azure Load Balancer에 대한 백end 주소 풀 구성을 만듭니다.

## 예제

### 예제 1: 부하 균형 조정에 대한 백end 주소 풀 구성 만들기
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

이 명령은 부하 균형 조정에 대한 BackendAddressPool02라는 백end 주소 풀 구성을 만듭니다.
