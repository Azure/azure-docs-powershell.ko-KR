# Get-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
애플리케이션 게이트웨이의 백 엔드 HTTP 설정을 얻습니다.

## 구문

```
Get-AzApplicationGatewayBackendHttpSetting [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
Get-AzApplicationGatewayBackendHttpSetting cmdlet은 애플리케이션 게이트웨이의 백 엔드 HTTP 설정을 얻습니다.

## 예제

### 예제 1: 이름에 따라 백 엔드 HTTP 설정 확인
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Settings  = Get-AzApplicationGatewayBackendHttpSetting -Name "Settings01" -ApplicationGateway $AppGw
```

첫 번째 명령은 ResourceGroup01이라는 리소스 그룹에 ApplicationGateway01이라는 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다. 두 번째 명령은 $AppGw Settings01이라는 HTTP 설정을 $Settings 변수에 저장합니다.

### 예제 2: 백 엔드 HTTP 설정 컬렉션을 확인합니다.
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SettingsList  = Get-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw
```

첫 번째 명령은 ResourceGroup01이라는 리소스 그룹에 ApplicationGateway01이라는 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다. 두 번째 명령은 $AppGw 대한 HTTP 설정 컬렉션을 $SettingsList 변수에 저장합니다.
