# Get-AzVirtualNetworkTap

## SYNOPSIS
가상 네트워크 탭을 얻습니다.

## 구문

### ListParameterSet(기본값)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명
**Get-AzVirtualNetworkTap** cmdlet은 Azure 가상 네트워크 탭 또는 리소스 그룹의 Azure 가상 네트워크 탭 목록을 얻습니다.

## 예제

### 예제 1: 가상 네트워크 탭
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

이 명령은 "ResourceGroup1"에서 주어진 "VirtualTap1"에 대한 VirtualNetwork 탭 참조를 얻습니다.

### 예제 2: 필터링을 사용하여 모든 가상 네트워크 탭을 얻습니다.
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

이 명령은 "VirtualTap"로 시작하는 모든 VirtualNetwork 탭 참조를 얻습니다.
