# Remove-AzApplicationGatewayIPConfiguration

## SYNOPSIS
애플리케이션 게이트웨이에서 IP 구성을 제거합니다.

## 구문

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Remove-AzApplicationGatewayIPConfiguration** cmdlet은 Azure 애플리케이션 게이트웨이에서 IP 구성을 제거합니다.

## 예제

### 예제 1: Azure 애플리케이션 게이트웨이에서 IP 구성 제거
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

첫 번째 명령은 애플리케이션 게이트웨이를 사용하여 애플리케이션 게이트웨이를 $AppGw 변수에 저장합니다.
두 번째 명령은 서브넷에 저장된 애플리케이션 게이트웨이에서 Subnet02라는 IP 구성을 $AppGw.
