# Get-AzRedisEnterpriseCacheOperationStatus

## SYNOPSIS
작업의 상태를 얻습니다.

## 구문

```
Get-AzRedisEnterpriseCacheOperationStatus -Location <String> -OperationId <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 설명
작업의 상태를 얻습니다.
