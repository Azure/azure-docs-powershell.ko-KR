# New-AzDeploymentManagerArtifactSource

## SYNOPSIS
아티팩트 원본을 만듭니다.

## 구문

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**New-AzDeploymentManagerArtifactSource** cmdlet은 아티팩트 원본을 만듭니다.
*이름,* *ResourceGroupName 및* 필수 속성을 지정합니다.

반환된 개체를 로컬로 수정한 다음, Set-AzDeploymentManagerArtifactSource cmdlet을 사용하여 아티팩트 원본에 변경 내용을 적용할 수 있습니다.

이 cmdlet은 ServiceUnit 리소스, 템플릿 및 매개 변수 파일에 필요한 아티팩트를 이 위치에서 참조할 수 있도록 New-AzDeploymentManagerServiceTopology cmdlet에서 참조할 수 있는 ResourceId가 있는 ArtifactSource 개체를 반환합니다.

## 예제

### 예제 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

ContosoResourceGroup에서 리소스의 위치로 미국 중부와 함께 ContosoArtifactSource 이름으로 아티팩트 원본을 만듭니다. SasUri 속성은 아티팩트가 저장되는 스토리지 컨테이너에 Azure Storage SAS URI를 제공합니다.
