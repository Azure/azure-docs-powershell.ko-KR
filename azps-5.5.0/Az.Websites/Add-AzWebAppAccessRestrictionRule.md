# Add-AzWebAppAccessRestrictionRule

## SYNOPSIS
Azure Web App에 Access Restiction 규칙을 추가합니다.

## 설명
**Add-AzWebAppAccessRestrictionRule** cmdlet은 Azure Web App에 액세스 제한 규칙을 추가합니다.

## 예제

### 예제 1: 웹앱에 IpAddress 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name IpRule -Priority 200 -Action Allow -IpAddress 10.10.0.0/8
```

이 명령은 우선 순위 200 및 ip 범위가 있는 액세스 제한 규칙을 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱에 추가합니다.

### 예제 2: 웹앱에 서브넷 서비스 엔드포인트 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name SubnetRule -Priority 300 -Action Allow -SubnetName appgw-subnet -VirtualNetworkName corp-vnet
```

이 명령은 우선 순위가 300인 액세스 제한 규칙을 corp-vnet의 서브넷 appgw-subnet을 사용하여 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱에 추가합니다.

### 예제 3: 웹앱에 ServiceTag 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name ServiceTagRule -Priority 200 -Action Allow -ServiceTag AzureFrontDoor.Backend
```

이 명령은 우선 순위가 200인 액세스 제한 규칙과 Azure Front Door의 IP 범위를 나타내는 서비스 태그를 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱에 추가합니다.

### 예제 4: 웹앱에 다중 주소 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name MultipleIpRule -Priority 200 -Action Allow -IpAddress "10.10.0.0/8,192.168.0.0/16"
```

이 명령은 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱에 우선 순위 200 및 두 개의 IP 범위가 있는 액세스 제한 규칙을 추가합니다.

### 예제 5: 웹앱에 http 헤더를 사용하는 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name MultipleIpRule -Priority 400 -Action Allow -ServiceTag AzureFrontDoor.Backend
-HttpHeader @{'x-forwarded-host' = 'www.contoso.com', 'app.contoso.com'; 'x-azure-fdid' = '355deb06-47c4-4ba4-9641-c7d7a98b913e'}
```

이 명령은 서비스 태그 AzureFrontDoor.Backend에 대해 우선 순위가 400인 액세스 제한 규칙을 추가하고, 특정 값의 http 헤더에 대한 액세스를 리소스 그룹 Default-Web-WestUS에 속하는 ContosoSite라는 웹앱으로만 추가로 제한합니다.
