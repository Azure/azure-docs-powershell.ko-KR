# Stop-AzBatchTask

## SYNOPSIS
Batch 작업을 중지합니다.

## 구문

### ID
```
Stop-AzBatchTask [-JobId] <String> [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObject
```
Stop-AzBatchTask [-Task] <PSCloudTask> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Stop-AzBatchTask** cmdlet은 Azure Batch 작업을 중지합니다.

## 예제

### 예제 1: ID로 Batch 작업 삭제
```
PS C:\>Stop-AzBatchTask -JobId "Job-000001" -Id "Task23" -BatchContext $Context
```

이 명령은 ID Job-000001이 있는 작업에서 ID Task23이 있는 태스크를 중지합니다.
이 명령은 확인을 요청하는 메시지를 제공합니다.
Get-AzBatchAccountKey cmdlet을 사용하여 변수 변수에 컨텍스트를 $Context.

### 예제 2: 파이프라인을 사용하여 Batch 작업 중지
```
PS C:\>Get-AzBatchTask -JobId "Job-000001" -Id "Task26" -BatchContext $Context | Stop-AzBatchTask -BatchContext $Context
```

이 명령은 Get-AzBatchTask cmdlet을 사용하여 ID Job-000001이 있는 작업에 ID Task26이 있는 Batch 태스크를 Get-AzBatchTask 합니다.
이 명령은 파이프라인 연산자를 사용하여 이 작업을 현재 cmdlet에 전달합니다.
이 명령은 이 작업을 중지합니다.
