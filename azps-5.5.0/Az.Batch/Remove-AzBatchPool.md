# Remove-AzBatchPool

## SYNOPSIS
지정된 Batch 풀을 삭제합니다.

## 구문

```
Remove-AzBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzBatchPool** cmdlet은 지정된 Azure Batch 풀을 삭제합니다.
Force 매개 변수를 사용하지 않는 한 확인 메시지가 *표시됩니다.*

## 예제

### 예제 1: 풀 ID로 Batch 풀 삭제
```
PS C:\>Remove-AzBatchPool -Id "MyPool" -BatchContext $Context
```

이 명령은 ID MyPool을 사용하여 풀을 삭제합니다.
삭제 작업이 수행되기 전에 사용자에게 확인 메시지가 표시됩니다.

### 예제 2: 강제로 모든 Batch 풀 삭제
```
PS C:\>Get-AzBatchPool -BatchContext $Context | Remove-AzBatchPool -Force -BatchContext $Context
```

이 명령은 모든 Batch 풀을 삭제합니다.
Force *매개* 변수가 있기 때문에 확인 프롬프트가 표시 안 됩니다.

이 cmdlet이 Batch 서비스와 상호 작용하는 데 사용하는 **BatchAccountContext** 인스턴스를 지정합니다.
