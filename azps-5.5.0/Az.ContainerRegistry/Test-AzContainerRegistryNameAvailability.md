# Test-AzContainerRegistryNameAvailability

## SYNOPSIS
컨테이너 레지스트리 이름의 가용성을 검사합니다.

## 구문

```
Test-AzContainerRegistryNameAvailability [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
이 Test-AzContainerRegistryNameAvailability cmdlet은 컨테이너 레지스트리 이름이 유효하고 사용할 수 있는지 여부를 검사합니다.

## 예제

### 예제 1: 컨테이너 레지스트리 이름의 가용성 확인
```powershell
PS C:\>Test-AzContainerRegistryNameAvailability -Name 'SomeRegistryName'

NameAvailable Reason Message
------------- ------ -------
         True
```

이 명령은 컨테이너 레지스트리 이름 \` SomeRegistryName의 가용성을 \` 검사합니다.
