# Add-AzSqlServerKeyVaultKey

## SYNOPSIS
Key Vault 키를 SQL 서버에 추가합니다.

## 구문

```
Add-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
Add-AzSqlServerKeyVaultKey cmdlet은 제공된 자격 증명 모음 서버에 Key Vault SQL 추가합니다.
서버에 자격 증명 모음에 대한 'get, wrapKey, unwrapKey' 권한이 있어야 합니다.

## 예제

### 예제 1: Key Vault 키 추가
```
PS C:\> Add-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

이 명령은 https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 'ContosoResourceGroup' 리소스 그룹의 'ContosoServer'라는 SQL 서버에 ID '가 있는 Key Vault 키를 추가합니다.
