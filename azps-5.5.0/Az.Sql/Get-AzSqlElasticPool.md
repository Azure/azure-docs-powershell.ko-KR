# Get-AzSqlElasticPool

## SYNOPSIS
Azure SQL Database에서 탄력적 풀 및 해당 속성 값을 얻습니다.

## 구문

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명
**Get-AzSqlElasticPool** cmdlet은 탄력적 풀 및 속성 값을 얻습니다.
기존 탄력적 풀의 이름을 지정하여 해당 풀에 대한 속성 값만 표시합니다.
