# Save-AzApiManagementTenantGitConfiguration

## SYNOPSIS
현재 구성에 대한 커밋을 만들어 변경 내용을 저장합니다.

## 구문

```
Save-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Save-AzApiManagementTenantGitConfiguration** cmdlet은 리포지토리의 분기에 현재 구성 스냅숏을 포함하는 커밋을 만들어 변경 내용을 저장합니다.

## 예제

### 예제 1: 구성에 변경 내용 저장
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Save-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

이 명령은 현재 구성 스냅숏을 사용하여 리포지토리의 지정된 분기에 커밋을 만들어 변경 내용을 저장합니다.
