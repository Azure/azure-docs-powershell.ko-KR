# Remove-AzDataLakeStoreItemAcl

## SYNOPSIS
Data Lake Store에서 파일 또는 폴더의 ACL을 지워야 합니다.

## 구문

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Remove-AzDataLakeStoreItemAcl** cmdlet은 Data Lake Store에 있는 파일 또는 폴더의 ACL(액세스 제어 목록)을 지우습니다.

## 예제

### 예제 1: 폴더에서 ACL 제거
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

이 명령은 ContosoADL 계정에 대한 루트 디렉터리에 대한 ACL을 제거합니다.
