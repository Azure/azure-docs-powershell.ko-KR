# Enable-AzDataLakeStoreKeyVault

## SYNOPSIS
지정된 Data Lake Store 계정의 암호화를 위해 사용자 관리 Key Vault를 사용하도록 설정하려고 시도합니다.

## 구문

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명
**Enable-AzDataLakeStoreKeyVault** cmdlet은 지정된 Data Lake Store 계정의 암호화를 위해 사용자 관리 Key Vault를 사용하도록 설정하려고 합니다.

## 예제

### 예제 1: ContosoADLS 계정에 Key Vault 사용
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

이 명령은 ContosoADLS라는 Data Lake Store 계정에 대해 사용자 관리 Key Vault를 사용하도록 설정하려고 합니다.
