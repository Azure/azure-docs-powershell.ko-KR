# Update-AzVpnServerConfiguration

## SYNOPSIS

## 설명은
**AzVpnServerConfiguration** cmdlet을 사용 하면 다른 VpnProtocols, VpnAuthenticationTypes, VpnServerConfiguration 정책을 사용 하 여 기존를 업데이트 하 고, 사이트 연결을 위한 고객의 요구 사항에 따라 선택한 vpn 인증 유형 관련 매개 변수를 설정할 수 있습니다.
