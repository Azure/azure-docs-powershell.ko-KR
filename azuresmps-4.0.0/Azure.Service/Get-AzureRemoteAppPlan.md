# Get-AzureRemoteAppPlan

## SYNOPSIS
Azure RemoteApp 계획을 나열합니다.

## 구문

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 설명
**Get-AzureRemoteAppPlan** cmdlet은 Azure RemoteApp 계획을 나열합니다.
